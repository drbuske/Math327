    Handout Daily Prep 2.3 - Higher Order Linear ODEs
    

    Section Overview

    In this section, we introduce the theory of higher‑order linear homogeneous differential equations, extending familiar second‑order ideas to the general \(n\)th‑order case. We learn that while most real‑world models are second order, the same foundational principles—linearity, superposition, and linear independence—carry over directly to higher order. The section emphasizes how solutions are built from sets of linearly independent functions and introduces the Wronskian as a computational tool for verifying independence, preparing us to analyze more complex systems using methods that generalize the second‑order theory.
    🔗
    🔗

      🔗
      🔗
    • Recognize the role of linear independence in constructing general solutions involving \(n\) fundamental functions.
      🔗
      🔗
    • Learn the definition and purpose of the Wronskian as a test for linear independence.

    Exercises Exercises

    1. Definition of Higher-Order Linear ODEs.
    Which of the following is a fourth‑order linear differential equation?
    🔗
    • \(y^{(4)} + p(x)y'' + q(x)y' + r(x)y = 0\)
    • Correct. An \(n\)th‑order linear ODE may be written as \(A_n(x)y^{(n)} + \dots + A_0(x)y = F(x)\text{,}\) with no nonlinear functions of \(y\text{.}\)
    • \(y^{(4)} + (y')^{2} = 0\)
    • Not linear: \((y')^{2}\) is nonlinear.
    • \(y'' + p(x)y = 0\)
    • This is linear but only second‑order, not higher‑order.
    🔗
    2. Superposition Principle.
    For the homogeneous equation \(L[y] = 0\text{,}\) where \(L\) is a linear differential operator of order \(n\text{,}\) which statement is true?
    🔗
    • If \(y_1,\dots,y_k\) are solutions, then any linear combination of them is also a solution.
    • Correct. Linear operators satisfy \(L(c_1y_1 + \dots + c_ky_k) = c_1L(y_1) + \dots + c_kL(y_k)\text{.}\)
    • The product \(y_1 y_2\) is always a solution.
    • No—products do not preserve linearity.
    • Only \(y_1\) can be used; other solutions are irrelevant.
    • No—higher‑order ODEs require multiple independent solutions.
    🔗
    3. Characteristic Polynomial for a 3rd-Order ODE.
    What is the characteristic polynomial for the ODE \(y''' - 2y'' + y' = 0\text{?}\)
    🔗
    • \(r^{3} - 2r^{2} + r = 0\)
    • Correct. Substituting \(y=e^{rx}\) gives \(r^3 - 2r^2 + r = 0\text{.}\)
    • \(r^{2} - 2r + 1 = 0\)
    • This is only quadratic; the ODE is third‑order.
    • \(r^{3} + r^{2} - 2r = 0\)
    • The sign of the \(r^2\) term is incorrect.
    🔗
    4. Independent Solutions and Order.
    How many linearly independent solutions are needed to describe the general solution of an \(n\)th‑order linear homogeneous ODE?
    🔗
    • Exactly \(n\) linearly independent solutions.
    • Correct. An \(n\)th‑order linear ODE requires \(n\) fundamental solutions to span the solution space.
    • Only one solution is needed.
    • No—this works only for first‑order equations.
    • Infinitely many solutions must be listed individually.
    • Incorrect—only \(n\) independent ones are required.
    🔗
    5. Solving a 3rd-Order ODE.
    Solve the homogeneous ODE \(y''' - 3y'' + 3y' = 0\text{.}\)
    🔗
    • \(y = C_1 + C_2 x + C_3 x^{2}\)
    • Correct. The characteristic polynomial is \(r^3 - 3r^2 + 3r = r(r-1)^2\text{,}\) giving roots \(0\) and \(1\) (double). Solutions are \(e^{0x}=1\text{,}\) \(e^{x}\text{,}\) and \(xe^{x}\text{;}\) rewritten in polynomial form, this yields \(C_1 + C_2 x + C_3 x^{2}\text{.}\)
    • \(y = C_1 e^{x} + C_2 e^{-x} + C_3 e^{2x}\)
    • Those roots do not satisfy the characteristic polynomial.
    • \(y = C_1 e^{x} + C_2 x e^{x}\)
    • Missing the third independent solution.
    🔗
    6.
    Find the general solution to \(y''' + 2y'' - y' - 2y = 0.\)
    🔗
    Solution.
    Write the characteristic equation:
    🔗
    \begin{equation*} r^{3} + 2r^{2} - r - 2 = 0. \end{equation*}
    Factor by grouping:
    🔗
    \begin{align*} r^{2}(r+2) - 1(r+2) \amp = (r^{2} - 1)(r+2) \\ \amp = (r-1)(r+1)(r+2). \end{align*}
    Thus the roots are \(r=1\text{,}\) \(r=-1\text{,}\) and \(r=-2\text{.}\) The general solution is:
    🔗
    \begin{equation*} y(x) = C_{1} e^{x} + C_{2} e^{-x} + C_{3} e^{-2x}. \end{equation*}
    🔗
    🔗
    7.
    Find the general solution to \(y^{(iv)} - y''' + 2y' = 0.\) Hint: The characteristic polynomial factors as \(r(r+1)(r^{2} - 2r + 2) = 0.\)
    🔗
    Solution.
    Using the factorization:
    🔗
    \begin{equation*} r(r+1)(r^{2} - 2r + 2) = 0. \end{equation*}
    The real roots are \(r=0\) and \(r=-1\text{.}\) The quadratic factor gives:
    🔗
    \begin{equation*} r^{2} - 2r + 2 = 0 \quad\Longrightarrow\quad r = 1 \pm i. \end{equation*}
    Thus the solution pieces are:
    🔗
    • \(\displaystyle e^{0x} = 1\)
      🔗
    • \(\displaystyle e^{-x}\)
      🔗
    • \(e^{x}\cos x\text{,}\) \(e^{x}\sin x\) (from \(1 \pm i\))
      🔗
    Therefore the general solution is:
    🔗
    \begin{equation*} y(x) = C_{1} + C_{2} e^{-x} + C_{3} e^{x} \cos x + C_{4} e^{x} \sin x. \end{equation*}
    🔗
    🔗
    8.
    Find the general solution to \(y^{(v)} + 18y''' + 81y' = 0.\) Hint: The characteristic polynomial factors as \(r(r^{2} + 9)^{2} = 0.\)
    🔗
    Solution.
    From the factorization:
    🔗
    \begin{equation*} r(r^{2}+9)^{2} = 0, \end{equation*}
    the roots are:
    🔗
    • \(\displaystyle r = 0\)
      🔗
    • \(r = 3i\) (double root)
      🔗
    • \(r = -3i\) (double root)
      🔗
    For \(r=0\text{,}\) the solution term is \(1\text{.}\) For the repeated complex roots \(\pm 3i\text{,}\) the general solution includes:
    🔗
    \begin{equation*} \cos 3x,\quad \sin 3x,\quad x\cos 3x,\quad x\sin 3x. \end{equation*}
    Therefore the general solution is:
    🔗
    \begin{equation*} y(x) = C_{1} + C_{2}\cos(3x) + C_{3}\sin(3x) + C_{4}x\cos(3x) + C_{5}x\sin(3x). \end{equation*}
    🔗
    🔗
    9.
    Consider the functions \(y_{1}(x) = e^{x}\text{,}\) \(y_{2}(x) = e^{2x}\text{,}\) and \(y_{3}(x) = e^{3x}\text{.}\) Compute their Wronskian \(W(y_{1},y_{2},y_{3})(x)\text{.}\) What does the result tell you about linear independence?
    🔗
    Solution.
    The Wronskian of three functions is
    🔗
    \begin{equation*} W = \begin{vmatrix} y_1 \amp y_2 \amp y_3 \\ y_1' \amp y_2' \amp y_3' \\ y_1'' \amp y_2'' \amp y_3'' \end{vmatrix} = \begin{vmatrix} e^{x} \amp e^{2x} \amp e^{3x} \\ e^{x} \amp 2e^{2x} \amp 3e^{3x} \\ e^{x} \amp 4e^{2x} \amp 9e^{3x} \end{vmatrix}. \end{equation*}
    Factor out \(e^{x}, e^{2x}, e^{3x}\) from each column:
    🔗
    \begin{equation*} W = e^{x} e^{2x} e^{3x} \begin{vmatrix} 1 \amp 1 \amp 1 \\ 1 \amp 2 \amp 3 \\ 1 \amp 4 \amp 9 \end{vmatrix} = e^{6x} \begin{vmatrix} 1 \amp 1 \amp 1 \\ 1 \amp 2 \amp 3 \\ 1 \amp 4 \amp 9 \end{vmatrix}. \end{equation*}
    Compute the 3×3 determinant:
    🔗
    \begin{align*} \begin{vmatrix} 1 \amp 1 \amp 1 \\ 1 \amp 2 \amp 3 \\ 1 \amp 4 \amp 9 \end{vmatrix} \amp = 1(2\cdot 9 - 3\cdot 4) -1(1\cdot 9 - 3\cdot 1) +1(1\cdot 4 - 2\cdot 1) \\ \amp = (18 - 12) - (9 - 3) + (4 - 2) \\ \amp = 6 - 6 + 2 \\ \amp = 2. \end{align*}
    Therefore:
    🔗
    \begin{equation*} W(y_{1},y_{2},y_{3})(x) = 2 e^{6x}. \end{equation*}
    Since \(W(x) \neq 0\) for all \(x\text{,}\) the functions are linearly independent on every interval. They therefore form a fundamental set of solutions for a third‑order linear homogeneous ODE.
    🔗
    🔗
    🔗
    10.
    Let \(y_{1}(x) = x\text{,}\) \(y_{2}(x) = x^{2}\text{,}\) \(y_{3}(x) = x^{3}\text{.}\) Compute the Wronskian \(W(y_{1},y_{2},y_{3})(x)\) and determine whether the functions are linearly independent. Are they valid as a fundamental solution set for a third‑order linear homogeneous ODE?
    🔗
    Solution.
    Compute the Wronskian:
    🔗
    \begin{equation*} W = \begin{vmatrix} x \amp x^{2} \amp x^{3} \\ 1 \amp 2x \amp 3x^{2} \\ 0 \amp 2 \amp 6x \end{vmatrix}. \end{equation*}
    Expand along the first row:
    🔗
    \begin{equation*} W = x \begin{vmatrix} 2x \amp 3x^{2} \\ 2 \amp 6x \end{vmatrix} - x^{2} \begin{vmatrix} 1 \amp 3x^{2} \\ 0 \amp 6x \end{vmatrix} + x^{3} \begin{vmatrix} 1 \amp 2x \\ 0 \amp 2 \end{vmatrix}. \end{equation*}
    Compute each 2×2 determinant:
    🔗
    \begin{align*} x[(2x)(6x) - (3x^{2})(2)] - x^{2}[6x] + x^{3}[2] \amp = x(12x^{2} - 6x^{2}) - 6x^{3} + 2x^{3} \\ \amp = x(6x^{2}) - 4x^{3} \\ \amp = 6x^{3} - 4x^{3} \\ \amp = 2x^{3}. \end{align*}
    Thus:
    🔗
    \begin{equation*} W(y_1,y_2,y_3)(x) = 2 x^{3}. \end{equation*}
    The Wronskian is zero only at \(x=0\text{,}\) but nonzero for \(x \neq 0\text{.}\) Therefore, the functions are linearly independent on any interval not containing 0 (e.g., \((0,\infty)\) or \((-\infty,0)\)).
    🔗
    Hence they form a valid fundamental solution set for a third‑order homogeneous linear ODE on any such interval.

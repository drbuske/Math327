    Exercises Questions

    1.
    For each of the following differential equations, classify it according to the following criteria:
    🔗
    • whether it is ordinary or partial,
      🔗
    • its order,
      🔗
    • whether it is linear or nonlinear,
      🔗
    • and, if applicable, whether it is homogeneous or non‑homogeneous.
      🔗
    Classify the differential equation:
    🔗
    \begin{equation*} y'' + 5y' - 6y = \sin x. \end{equation*}
    Hint.
    Recall the key definitions:
    🔗
    • An ordinary differential equation (ODE) involves a function of a single variable and its ordinary derivatives.
      🔗
    • The order is the highest derivative present.
      🔗
    • A differential equation is linear if the dependent variable and all its derivatives appear to the first power and are not multiplied together.
      🔗
    • A linear differential equation is homogeneous if the right-hand side is zero; otherwise it is non-homogeneous.
      🔗
    🔗
    Answer.
    The equation is an ordinary differential equation of second order, it is linear, and it is non-homogeneous.
    🔗
    🔗
    Solution.
    We analyze the equation
    🔗
    \begin{equation*} y'' + 5y' - 6y = \sin x. \end{equation*}
    First, because the unknown function is \(y(x)\text{,}\) a function of a single variable, and all derivatives are ordinary derivatives with respect to \(x\text{,}\)this is an ordinary differential equation.
    🔗
    The highest derivative appearing is \(y''\text{,}\) so the order of the equation is \(2\text{.}\)
    🔗
    Each occurrence of \(y\text{,}\) \(y'\text{,}\) and \(y''\) appears to the first power, and none are multiplied together or appear inside nonlinear expressions such as trigonometric, exponential, or power functions. Therefore the equation is linear.
    🔗
    A linear differential equation is homogeneous only when the right-hand side is identically zero. Here the right-hand side is \(\sin x\text{,}\) which is not zero; hence the equation is non-homogeneous.
    🔗
    Putting everything together: the equation is an ordinary, second-order, linear, non-homogeneous differential equation.
    🔗
    🔗
    🔗
    2.
    Consider the differential equation
    🔗
    \begin{equation*} y'(x) = 3x^2 - 4x + 1. \end{equation*}
    1. Find the general solution using an integral expression.
      🔗
    2. Determine the particular solution satisfying the initial condition \(y(1)=2\text{.}\)
      🔗
    🔗
    Hint.
    Recall that if \(y'(x)=f(x)\text{,}\) then any solution must satisfy
    🔗
    \begin{equation*} y(x) = \int f(x)\,dx + C. \end{equation*}
    Integrate term‑by‑term. After finding the general solution, substitute \(x=1\) and \(y=2\) to determine \(C\text{.}\)
    🔗
    🔗
    Answer.
    The general solution is \(y(x)=x^3 - 2x^2 + x + C\text{.}\) The initial condition \(y(1)=2\) gives \(C=2\text{,}\) so the particular solution is \(y(x)=x^3 - 2x^2 + x + 2\text{.}\)
    🔗
    🔗
    Solution.
    We begin with the differential equation
    🔗
    \begin{equation*} y'(x) = 3x^2 - 4x + 1. \end{equation*}
    By the fundamental theorem of calculus, a solution must satisfy
    🔗
    \begin{equation*} y(x) = \int (3x^2 - 4x + 1)\,dx + C. \end{equation*}
    Integrating term‑by‑term:
    🔗
    \begin{equation*} \int 3x^2\,dx = x^3,\qquad \int (-4x)\,dx = -2x^2,\qquad \int 1\,dx = x. \end{equation*}
    Therefore the general solution is
    🔗
    \begin{equation*} y(x) = x^3 - 2x^2 + x + C. \end{equation*}
    To find the particular solution satisfying \(y(1)=2\text{,}\) substitute \(x=1\) into the general solution:
    🔗
    \begin{equation*} 2 = 1^3 - 2\cdot 1^2 + 1 + C = 1 - 2 + 1 + C = 0 + C. \end{equation*}
    Thus \(C = 2\text{,}\) and the particular solution is
    🔗
    \begin{equation*} y(x) = x^3 - 2x^2 + x + 2. \end{equation*}
    🔗
    🔗
    3.
    Consider the differential equation
    🔗
    \begin{equation*} y' = x - y. \end{equation*}
    Sketch a small portion of the slope field in the region \(-1 \le x \le 2\text{,}\) \(-1 \le y \le 2\) by hand. Then answer the following questions:
    🔗
    1. Determine whether the solution curve through the point \((0,1)\) is initially increasing or decreasing. Justify your answer using the slope field.
      🔗
    2. Without solving the differential equation explicitly, describe the qualitative long–term behavior of solutions. Do they tend toward a particular line, diverge, or behave in some other way?
      🔗
    Hint.
    Remember that the slope field is built from small line segments whose slope equals \(y' = x - y\) at each point \((x,y)\text{.}\) For part (a), compute \(y'\) at the given point to determine whether the local slope is positive or negative. For part (b), think about values of \(x - y\) along the line \(y = x-1\) and whether slopes push trajectories toward or away from that line.
    🔗
    🔗
    Answer.
    (a) The solution through \((0,1)\) is initially decreasing because \(y' = 0 - 1 = -1\text{.}\) (b) Solutions tend toward the line \(y = x-1\text{,}\) which acts as a stable equilibrium line for the differential equation.
    🔗
    🔗
    Solution.
    We are given the differential equation
    🔗
    \begin{equation*} y' = x - y. \end{equation*}
    To understand the qualitative behavior of solutions, we begin by noting that the slope field at each point \((x,y)\) is determined by the value of \(x - y\text{.}\) In particular:
    🔗
    • When \(x - y = 0\text{,}\) the slope is horizontal.
      🔗
    • When \(x - y > 0\text{,}\) solution curves locally increase.
      🔗
    • When \(x - y < 0\text{,}\) solution curves locally decrease.
      🔗
    (a) At the point \((0,1)\) we compute
    🔗
    \begin{equation*} y' = x - y = 0 - 1 = -1. \end{equation*}
    Because the slope is negative, the solution curve passing through \((0,1)\) initially decreases as \(x\) increases.
    🔗
    (b) To analyze long–term behavior, we solve the differential equation. Rewriting it in linear form,
    🔗
    \begin{equation*} y' + y = x, \end{equation*}
    the integrating factor is \(e^x\text{,}\) and multiplying both sides yields
    🔗
    \begin{equation*} (e^x y)' = x e^x. \end{equation*}
    Integrating,
    🔗
    \begin{equation*} e^x y = x e^x - e^x + C, \end{equation*}
    and dividing by \(e^x\) gives the general solution
    🔗
    \begin{equation*} y(x) = x - 1 + C e^{-x}. \end{equation*}
    As \(x \to \infty\text{,}\) the exponential term \(C e^{-x}\) decays to zero. Therefore every solution approaches the straight line
    🔗
    \begin{equation*} y = x - 1, \end{equation*}
    which serves as the asymptotic line that solutions follow for large \(x\text{.}\) In particular, this line is an attracting solution curve: trajectories above it are pushed downward, and trajectories below it are pushed upward.
    🔗
    The qualitative picture is that all solutions to \(y' = x - y\) approach the line \(y = x - 1\) as \(x\) increases.
    🔗
    🔗
    🔗
    4.
    Consider the differential equation
    🔗
    \begin{equation*} \dfrac{dy}{dx} = x(y+1). \end{equation*}
    1. Separate variables and find the general solution.
      🔗
    2. Find the particular solution satisfying the initial condition \(y(0) = 2\text{.}\)
      🔗
    🔗
    Hint.
    First rewrite the equation so that all \(y\)-terms are on one side and all \(x\)-terms on the other:
    🔗
    \begin{equation*} \dfrac{1}{y+1}\,dy = x\,dx. \end{equation*}
    Then integrate both sides. After integrating, exponentiate to solve for \(y\text{,}\) and finally use the initial condition to determine the constant.
    🔗
    🔗
    Answer.
    The general solution is
    \begin{equation*} y = C e^{x^{2}/2} - 1. \end{equation*}
    The initial condition \(y(0)=2\) gives \(C=3\text{,}\) so the particular solution is
    \begin{equation*} y = 3 e^{x^{2}/2} - 1. \end{equation*}
    🔗
    🔗
    Solution.
    Starting with the differential equation
    🔗
    \begin{equation*} \dfrac{dy}{dx} = x(y+1), \end{equation*}
    we separate variables by dividing both sides by \((y+1)\) and multiplying by \(dx\text{:}\)
    🔗
    \begin{equation*} \dfrac{1}{y+1}\,dy = x\,dx. \end{equation*}
    Integrate both sides:
    🔗
    \begin{equation*} \int \dfrac{1}{y+1}\,dy = \int x\,dx. \end{equation*}
    Compute the integrals:
    🔗
    \begin{equation*} \ln|y+1| = \dfrac{x^{2}}{2} + C. \end{equation*}
    Exponentiating,
    🔗
    \begin{equation*} |y+1| = e^{\,x^{2}/2 + C}. \end{equation*}
    Write \(e^{C}\) as a new positive constant \(C_1\text{,}\) giving
    🔗
    \begin{equation*} y + 1 = C_1 e^{x^{2}/2}. \end{equation*}
    Solving for \(y\) produces the general solution:
    🔗
    \begin{equation*} y = C_1 e^{x^{2}/2} - 1. \end{equation*}
    Apply the initial condition \(y(0)=2\text{:}\)
    🔗
    \begin{equation*} 2 = C_1 e^{0} - 1 = C_1 - 1. \end{equation*}
    Hence \(C_1 = 3\text{,}\) and the particular solution is
    🔗
    \begin{equation*} y(x) = 3 e^{x^{2}/2} - 1. \end{equation*}
    🔗
    🔗
    5.
    Consider the linear differential equation
    🔗
    \begin{equation*} y' - 3y = 6e^{2x}. \end{equation*}
    1. Find the general solution using the integrating factor method.
      🔗
    2. Find the particular solution satisfying the initial condition \(y(0)=4\text{.}\)
      🔗
    🔗
    Hint.
    Rewrite the equation in the standard linear form
    \begin{equation*} y' + p(x)y = q(x) \end{equation*}
    and identify \(p(x)=-3\text{.}\) The integrating factor is
    🔗
    \begin{equation*} \mu(x) = e^{\int -3\,dx} = e^{-3x}. \end{equation*}
    Multiply both sides of the equation by this integrating factor. The left-hand side becomes \((e^{-3x}y)'\text{.}\) Integrate both sides, then multiply by \(e^{3x}\) to solve for \(y\text{.}\)
    🔗
    🔗
    Answer.
    The general solution is
    \begin{equation*} y(x) = -6e^{2x} + C e^{3x}. \end{equation*}
    Using \(y(0)=4\) gives \(C=10\text{,}\) so the particular solution is
    \begin{equation*} y(x) = -6e^{2x} + 10 e^{3x}. \end{equation*}
    🔗
    🔗
    Solution.
    We begin with the linear differential equation
    🔗
    \begin{equation*} y' - 3y = 6e^{2x}. \end{equation*}
    In linear form \(y' + p(x)y = q(x)\text{,}\) we have \(p(x)=-3\) and \(q(x)=6e^{2x}\text{.}\)
    🔗
    The integrating factor is
    🔗
    \begin{equation*} \mu(x) = e^{\int -3\,dx} = e^{-3x}. \end{equation*}
    Multiply the entire differential equation by \(e^{-3x}\text{:}\)
    🔗
    \begin{equation*} e^{-3x} y' - 3 e^{-3x} y = 6 e^{2x} e^{-3x}. \end{equation*}
    The left-hand side is the derivative of \(e^{-3x}y\text{:}\)
    🔗
    \begin{equation*} (e^{-3x}y)' = 6 e^{-x}. \end{equation*}
    Integrate both sides:
    🔗
    \begin{equation*} e^{-3x} y = \int 6 e^{-x}\,dx = -6 e^{-x} + C. \end{equation*}
    Multiply both sides by \(e^{3x}\) to solve for \(y\text{:}\)
    🔗
    \begin{equation*} y = -6 e^{-x} e^{3x} + C e^{3x}. \end{equation*}
    Simplifying the exponential:
    🔗
    \begin{equation*} y(x) = -6 e^{2x} + C e^{3x}. \end{equation*}
    To find the constant \(C\text{,}\) apply the initial condition \(y(0)=4\text{:}\)
    🔗
    \begin{equation*} 4 = -6 e^{0} + C e^{0} = -6 + C. \end{equation*}
    Thus \(C = 10\text{.}\)
    🔗
    Therefore the particular solution is
    🔗
    \begin{equation*} y(x) = -6 e^{2x} + 10 e^{3x}. \end{equation*}
    🔗
    🔗
    6.
    Consider the autonomous differential equation
    🔗
    \begin{equation*} x' = (1 - x)(3 - x)^2. \end{equation*}
    1. Find all equilibrium solutions.
      🔗
    2. Construct a phase diagram for this differential equation.
      🔗
    3. Classify each equilibrium as stable, unstable, or semi-stable based on your phase diagram.
      🔗
    4. Describe the long-term behavior of solutions with \(x(0)=0\text{,}\) \(x(0)=2\text{,}\) and \(x(0)=4\text{.}\)
      🔗
    🔗
    Hint.
    Compute the sign of \(f(x) = (1-x)(3-x)^2\) in each region:
    🔗
    • \(\displaystyle x < 1\)
      🔗
    • \(\displaystyle 1 < x < 3\)
      🔗
    • \(\displaystyle x > 3\)
      🔗
    Note that \((3-x)^2 \ge 0\) for all \(x\text{,}\) and equals zero only when \(x=3\text{.}\) So the sign of \(f(x)\) is entirely determined by \((1-x)\text{.}\)
    🔗
    Stability comes from arrow directions:
    🔗
    • Arrows pointing toward the equilibrium implies stable.
      🔗
    • Arrows pointing away implies unstable.
      🔗
    • Arrows pointing toward on one side and away on the other implies semi-stable.
      🔗
    🔗
    Answer.
    Equilibria: \(x = 1\) and \(x = 3\text{.}\) The phase diagram has arrows up for \(x<1\text{,}\) arrows down for \(1<x<3\text{,}\) and arrows down for \(x>3\text{.}\) Thus \(x=1\) is stable, and \(x=3\) is semi-stable. Long-term behavior:
    • \(x(0)=0\text{:}\) solution increases toward \(x=1\) from below.
      🔗
    • \(x(0)=2\text{:}\) solution decreases toward \(x=1\) from above.
      🔗
    • \(x(0)=4\text{:}\) solution decreases toward \(x=3\text{.}\)
      🔗
    🔗
    🔗
    Solution.
    (a) Equilibria occur where
    🔗
    \begin{equation*} (1-x)(3-x)^2 = 0, \end{equation*}
    giving \(x = 1\) and \(x = 3\text{.}\)
    🔗
    (b) To determine the sign of \(f(x) = (1-x)(3-x)^2\text{,}\) note that the squared factor \((3-x)^2\) is always nonnegative and equals zero only at \(x = 3\text{.}\) Therefore the sign of \(f(x)\) is the sign of \((1-x)\text{.}\)
    🔗
    • If \(x < 1\text{,}\) then \(1-x > 0\text{,}\) so \(x' > 0\text{.}\) Arrow: up.
      🔗
    • If \(1 < x < 3\text{,}\) then \(1-x < 0\text{,}\) so \(x' < 0\text{.}\) Arrow: down.
      🔗
    • If \(x > 3\text{,}\) then \(1-x < 0\text{,}\) so \(x' < 0\text{.}\) Arrow: down.
      🔗
    (c) Therefore the phase diagram is as in Figure 34.
    🔗
    Stability analysis:
    🔗
    • \(x = 1\text{:}\) arrows point towards \(x=1\) on both sides so the equilibrium is stable.
      🔗
    • \(x = 3\text{:}\) arrow from below points away from \(x=3\text{,}\) but arrow from above points downward towards \(x=3\text{.}\) Thus \(x=3\) is semi-stable equilibrium.
      🔗
    (d) Long–term behavior:
    🔗
    • If \(x(0)=0\text{,}\) then the arrow is upward; the solution moves toward the stable equilibrium \(x = 1\text{.}\)
      🔗
    • If \(x(0)=2\text{,}\) the arrow is downward; the solution decreases toward the stable equilibrium \(x=1\text{.}\)
      🔗
    • If \(x(0)=4\text{,}\) the arrow is downward; the solution decreases toward the equilibrium \(x=3\text{.}\)
      🔗

    7.
    Consider the initial value problem
    🔗
    \begin{equation*} y' = x + y, \qquad y(0) = 1. \end{equation*}
    Use Euler’s method with step size \(h = 0.1\) to approximate the value of \(y(0.3)\text{.}\) Show your intermediate computations for each Euler step.
    🔗
    Hint.
    Recall that Euler’s method updates the approximate solution by
    🔗
    \begin{equation*} y_{n+1} = y_n + h \, f(x_n, y_n), \end{equation*}
    where \(f(x,y) = x + y\text{.}\) Starting at \((x_0,y_0) = (0,1)\) and using \(h=0.1\text{,}\) compute successively:
    🔗
    • \(y_1\) at \(x_1 = 0.1\text{,}\)
      🔗
    • \(y_2\) at \(x_2 = 0.2\text{,}\)
      🔗
    • \(y_3\) at \(x_3 = 0.3\text{.}\)
      🔗
    🔗
    Answer.
    Using Euler’s method with \(h=0.1\text{,}\) \(y(0.3)\) is approximately
    \begin{equation*} y_3 = 1.462. \end{equation*}
    🔗
    🔗
    Solution.
    We apply Euler’s method with \(f(x,y) = x + y\) and \(h=0.1\text{.}\) The starting point is
    🔗
    \begin{equation*} x_0 = 0, \qquad y_0 = 1. \end{equation*}
    Step 1: Compute \(y_1\) at \(x_1 = 0.1\text{.}\)
    🔗
    \begin{equation*} f(x_0,y_0) = 0 + 1 = 1, \end{equation*}
    \begin{equation*} y_1 = y_0 + h f(x_0,y_0) = 1 + 0.1(1) = 1.1. \end{equation*}
    Step 2: Compute \(y_2\) at \(x_2 = 0.2\text{.}\)
    🔗
    \begin{equation*} f(x_1,y_1) = 0.1 + 1.1 = 1.2, \end{equation*}
    \begin{equation*} y_2 = y_1 + h f(x_1,y_1) = 1.1 + 0.1(1.2) = 1.22. \end{equation*}
    Step 3: Compute \(y_3\) at \(x_3 = 0.3\text{.}\)
    🔗
    \begin{equation*} f(x_2,y_2) = 0.2 + 1.22 = 1.42, \end{equation*}
    \begin{equation*} y_3 = y_2 + h f(x_2,y_2) = 1.22 + 0.1(1.42) = 1.22 + 0.142 = 1.462. \end{equation*}
    Therefore, the Euler approximation of \(y(0.3)\) with \(h=0.1\) is
    🔗
    \begin{equation*} y(0.3) \approx 1.462. \end{equation*}

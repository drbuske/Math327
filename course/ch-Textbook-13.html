    Handout Daily Prep 2.5 - Nonhomogeneous Equations
    

    Section Overview

    In this section, we learn how to solve nonhomogeneous linear differential equations by building on our knowledge of homogeneous equations. The core idea is that the general solution to a nonhomogeneous equation is obtained by adding any particular solution to the complementary (homogeneous) solution. We then explore why any two particular solutions differ by a homogeneous solution, and introduce the method of undetermined coefficients as a practical way to guess particular solutions when the forcing function has a familiar algebraic form.
    🔗
    🔗

    Exercises Exercises

    1.
    \(y_{c}\) is called a complementary solution. What do you think it complements?
    🔗
    Solution.
    The complementary solution \(y_c\) solves the associated homogeneous equation \(L[y]=0\text{.}\) It is called “complementary” because it complements a single particular solution \(y_p\) to produce the full general solution to the nonhomogeneous equation:
    🔗
    \begin{equation*} y = y_c + y_p. \end{equation*}
    In other words, \(y_c\) fills in all the homogeneous pieces required to describe every possible solution, complementing the specific contribution coming from the forcing term.
    🔗
    🔗
    🔗
    2.
    In your own words, summarize Theorem 2.5.1.
    🔗
    Solution.
    Theorem 2.5.1 says that to solve a linear nonhomogeneous ODE \(L[y]=f(x)\text{:}\)
    🔗
    • Find \(y_c\text{,}\) the general solution of the homogeneous equation \(L[y]=0\text{.}\)
      🔗
      🔗
    • Find \(y_p\text{,}\) any single particular solution of \(L[y]=f(x)\text{.}\)
      🔗
      🔗
    Then the full general solution is simply
    🔗
    \begin{equation*} y = y_c + y_p. \end{equation*}
    The theorem also guarantees that any two solutions differ by a homogeneous solution, so this formula captures all solutions.
    🔗
    🔗
    🔗
    3.
    Subsection 2.5.2, on Undetermined Coefficients, mentions a hiccup. What is this hiccup? Can it be managed? If so, how?
    🔗
    Solution.
    The “hiccup” occurs when the forcing term is itself a solution of the homogeneous equation. In that case, the usual trial function for \(y_p\) gives zero when substituted into the left‑hand side.
    🔗
    Yes, it can be managed: multiply the entire trial solution by \(x\) (or by \(x^2\) if one multiplication still conflicts) until the guess is no longer a solution of the homogeneous equation. This ensures a valid particular solution.
    🔗
    🔗
    🔗
    4.
    Give an example of a second‑order, linear, constant‑coefficient, nonhomogeneous ODE for which Undetermined Coefficients will not work.
    🔗
    Solution.
    Undetermined coefficients works only for forcing terms that are exponentials, polynomials, sines, and cosines (or finite sums/products of these).
    🔗
    A forcing term like \(\ln x\) or \(\tan x\) does not fit that pattern. For example:
    🔗
    \begin{equation*} y'' + 4y = \ln x \end{equation*}
    cannot be solved by undetermined coefficients. We would need variation of parameters instead.
    🔗
    🔗
    🔗
    5. Structure of the General Solution.
    For a nonhomogeneous linear ODE \(L[y] = f(x)\text{,}\) which formula describes the general solution?
    🔗
    • \(y = y_c + y_p\text{,}\) where \(y_c\) solves \(L[y] = 0\) and \(y_p\) is any particular solution.
    • Correct. Every solution is the sum of the complementary solution and any particular solution.
    • \(y = y_c \cdot y_p\)
    • No—solutions are added, not multiplied.
    • \(y = y_p\) only
    • No—the complementary solution is needed for the full family of solutions.
    🔗
    6. Why the Method Works.
    Suppose \(y_{p_1}\) and \(y_{p_2}\) are both particular solutions to \(L[y] = f(x)\text{.}\) What can be said about \(w = y_{p_1} - y_{p_2}\text{?}\)
    🔗
    • \(w\) solves the associated homogeneous equation \(L[w] = 0\text{.}\)
    • Correct. The difference of two particular solutions always satisfies \(L[w]=0\text{.}\)
    • \(w\) is another particular solution.
    • No—its right‑hand side is zero, not \(f(x)\text{.}\)
    • \(w = 0\) always.
    • Not true—different particular solutions differ by a homogeneous solution, not necessarily zero.
    🔗
    7. Undetermined Coefficients for a Polynomial.
    Consider \(y'' + 5y' + 6y = 2x + 1\text{.}\) Using undetermined coefficients, what form should you try for \(y_p\text{?}\)
    🔗
    • \(y_p = Ax + B\)
    • Correct. When \(f(x)\) is a polynomial, we try a polynomial of the same degree.
    • \(y_p = Ce^{x}\)
    • No—that form is for exponential forcing.
    • \(y_p = A\cos x + B\sin x\)
    • No—this corresponds to sinusoidal forcing.
    🔗
    8. Complementary Solution Identification.
    For the equation \(y'' + 5y' + 6y = 2x + 1\text{,}\) what is the complementary solution \(y_c\text{?}\)
    🔗
    • \(y_c = C_1 e^{-2x} + C_2 e^{-3x}\)
    • Correct. Solve \(r^2 + 5r + 6 = 0\) → \((r+2)(r+3)=0\text{.}\)
    • \(y_c = Ax + B\)
    • No—that form is for a polynomial forcing, not the homogeneous solution.
    • \(y_c = C_1 \cos x + C_2 \sin x\)
    • No—this form corresponds to complex roots.
    🔗
    9. Determining a Particular Solution Coefficient.
    In solving \(y'' + 5y' + 6y = 2x + 1\text{,}\) suppose you try \(y_p = Ax + B\text{.}\) Plugging into the left side gives \((6A)x + (5A + 6B)\text{.}\) What must \(A\) equal?
    🔗
    • \(A = \tfrac{1}{3}\)
    • Correct, because matching coefficients requires \(6A = 2\text{.}\)
    • \(A = 2\)
    • No—this would give \(6A = 12\text{,}\) not \(2\text{.}\)
    • \(A = \tfrac12\)
    • No—plugging in yields \(6A = 3\text{,}\) not \(2\text{.}\)
    🔗
    10. Avoiding Resonance in Undetermined Coefficients.
    Consider the ODE \(y'' + 4y = \cos(2x).\) When using the method of undetermined coefficients, what form should you try for the particular solution \(y_p\text{?}\)
    🔗
    • \(y_p = x\,(A\cos 2x + B\sin 2x)\)
    • Correct. Because \(\cos 2x\) and \(\sin 2x\) already solve the homogeneous equation (the system is in resonance), we multiply the usual guess by \(x\text{.}\) We shift the guess when it conflicts with the complementary solution.
    • \(y_p = A\cos 2x + B\sin 2x\)
    • No—this duplicates the complementary solution, yielding no new information.
    • \(y_p = A x^2 + Bx + C\)
    • No—this form is used for polynomial forcing, not trigonometric forcing.
    🔗
    11. Choosing a Particular Solution for an Exponential Forcing.
    Consider the equation \(y'' - 3y' + 2y = e^{x}.\) The complementary solution is \(y_c = C_1 e^{x} + C_2 e^{2x}\text{.}\) What is the correct form for a particular solution \(y_p\) using the method of undetermined coefficients?
    🔗
    • \(y_p = A x e^{x}\)
    • Correct. Because \(e^{x}\) is already part of the complementary solution, the trial function \(A e^{x}\) would duplicate a homogeneous solution and produce only the zero result. In this situation the trial form must be multiplied by \(x\) to avoid resonance, giving \(y_p = A x e^{x}\text{.}\)
    • \(y_p = A e^{x}\)
    • No—\(e^{x}\) already appears in \(y_c\text{,}\) so this choice leads to duplication and cannot produce a valid particular solution.
    • \(y_p = A e^{2x}\)
    • No—\(e^{2x}\) also appears in the complementary solution and is unrelated to the forcing term.
    🔗
    12. Finding a Particular Solution Quickly.
    For the equation \(y'' + y = 3e^{x},\) which of the following is a correct particular solution \(y_p\text{?}\)
    🔗
    • \(y_p = \tfrac{3}{2} e^{x}\)
    • Correct. Substituting \(y_p = Ae^{x}\) yields \(Ae^{x} + Ae^{x} = 3e^{x}\text{,}\) so \(2A=3 \longrightarrow A=\tfrac{3}{2}\text{.}\)
    • \(y_p = 3e^{x}\)
    • No—this gives \(y_p'' + y_p = 6e^{x}\text{,}\) which is too large.
    • \(y_p = \tfrac{3}{4}e^{x}\)
    • No—this yields only \(\tfrac{3}{2} e^{x}\) on substitution, not \(3e^{x}\text{.}\)
    🔗
    13.
    Find a particular solution of
    🔗
    \begin{equation*} 3y'' + y' - 2y = 2\cos x. \end{equation*}
    A reasonable trial is \(y_p = A\cos x + B\sin x\text{.}\)
    🔗
    Solution.
    Compute derivatives:
    🔗
    \begin{equation*} y_p' = -A\sin x + B\cos x, \qquad y_p'' = -A\cos x - B\sin x. \end{equation*}
    Substitute into the ODE:
    🔗
    \begin{equation*} 3(-A\cos x - B\sin x) + (-A\sin x + B\cos x) - 2(A\cos x + B\sin x) = 2\cos x. \end{equation*}
    Group cosine and sine terms:
    🔗
    \begin{equation*} \cos x\,(-3A + B - 2A) \;+\; \sin x\,(-3B - A - 2B) = 2\cos x. \end{equation*}
    So
    🔗
    \begin{equation*} (-5A + B)\cos x + (-6B - A)\sin x = 2\cos x. \end{equation*}
    Match coefficients:
    🔗
    \begin{equation*} -5A + B = 2, \qquad -A - 6B = 0. \end{equation*}
    Solve the second for \(A=-6B\text{.}\) Plug into the first:
    🔗
    \begin{equation*} -5(-6B) + B = 2 \quad\Rightarrow\quad 31B = 2 \quad\Rightarrow\quad B = \frac{2}{31}. \end{equation*}
    Then \(A = -6B = -\frac{12}{31}\text{.}\)
    🔗
    Thus a particular solution is
    🔗
    \begin{equation*} y_p(x) = -\frac{12}{31}\cos x + \frac{2}{31}\sin x. \end{equation*}
    🔗
    🔗
    14.
    Find a particular solution of
    🔗
    \begin{equation*} y'' - 4y = 2e^{2x}. \end{equation*}
    Since \(e^{2x}\) is a homogeneous solution, try \(y_p = A x e^{2x}\text{.}\)
    🔗
    Solution.
    Let \(y_p = A x e^{2x}\text{.}\) Compute:
    🔗
    \begin{equation*} y_p' = A e^{2x} + 2Ax e^{2x}, \qquad y_p'' = 2A e^{2x} + 4Ax e^{2x} + 2A e^{2x}. \end{equation*}
    Simplify:
    🔗
    \begin{equation*} y_p'' = 4A e^{2x} + 4Ax e^{2x}. \end{equation*}
    Now compute \(y_p'' - 4y_p\text{:}\)
    🔗
    \begin{equation*} y_p'' - 4y_p = (4A e^{2x} + 4Ax e^{2x}) - 4(Ax e^{2x}) = 4A e^{2x}. \end{equation*}
    Set \(4A e^{2x} = 2 e^{2x}\text{,}\) so \(A = \tfrac12\text{.}\) Therefore
    🔗
    \begin{equation*} y_p(x) = \frac12\, x e^{2x}. \end{equation*}
    🔗
    🔗
    15.
    1. Determine the appropriate form of a particular solution of
      🔗
      🔗
    🔗
    \begin{equation*} y'' + 6y' + 13y = e^{-3x}\cos 2x. \end{equation*}
    1. Find the coefficients in your trial solution.
      🔗
      🔗
    🔗
    Solution.
    1. The forcing term is \(e^{-3x}\cos 2x\text{.}\) The complementary solution comes from roots of
      🔗
      \begin{equation*} r^2 + 6r + 13 = 0 \quad\Rightarrow\quad r = -3 \pm 2i. \end{equation*}
      Since the forcing term has the same exponential \(e^{-3x}\) and same frequency \(2\text{,}\) it exactly matches the homogeneous form— so we must multiply by \(x\text{.}\)
      🔗
      Appropriate form:
      🔗
      \begin{equation*} y_p = x e^{-3x}\bigl(A\cos 2x + B\sin 2x\bigr). \end{equation*}
      🔗
    2. Differentiate and substitute (CAS recommended). The resulting coefficients are:
      🔗
      \begin{equation*} A = 0,\qquad B = -\frac14. \end{equation*}
      Hence a particular solution is
      🔗
      \begin{equation*} y_p(x) = -\frac14\, x e^{-3x}\sin 2x. \end{equation*}
      🔗
    🔗
    🔗
    16.
    Find a particular solution of
    🔗
    \begin{equation*} y'' - 4y' + 4y = e^{2x}. \end{equation*}
    Solution.
    Characteristic equation: \(r^2 - 4r + 4 = (r-2)^2\text{.}\) Since the forcing term is \(e^{2x}\text{,}\) which corresponds to the repeated root, try:
    🔗
    \begin{equation*} y_p = A x^2 e^{2x}. \end{equation*}
    Compute derivatives:
    🔗
    \begin{equation*} y_p' = A(2x e^{2x} + 2x^2 e^{2x}),\\ y_p'' = A(2e^{2x} + 8x e^{2x} + 4x^2 e^{2x}). \end{equation*}
    Substitute:
    🔗
    \begin{align*} y_p'' - 4y_p' + 4y_p \amp = A e^{2x}(2 + 8x + 4x^2) - 4A e^{2x}(2x + 2x^2) + 4A x^2 e^{2x} \\ \amp = 2A e^{2x}. \end{align*}
    Therefore set \(2A = 1\text{,}\) so \(A = \tfrac12\text{.}\) Thus
    🔗
    \begin{equation*} y_p = \tfrac12 x^2 e^{2x}. \end{equation*}
    🔗
    🔗
    17.
    Solve the initial value problem
    🔗
    \begin{equation*} y'' + 9y = \cos 3x + \sin 3x,\qquad y(0)=2,\quad y'(0)=1. \end{equation*}
    Solution.
    Complementary solution:
    🔗
    \begin{equation*} r^2 + 9 = 0 \quad\Rightarrow\quad r = \pm 3i. \end{equation*}
    So
    🔗
    \begin{equation*} y_c = C_1 \cos 3x + C_2 \sin 3x. \end{equation*}
    For a particular solution, try \(y_p = A\cos 3x + B\sin 3x\text{.}\) Substitute:
    🔗
    \begin{equation*} y_p'' + 9y_p = (-9A\cos 3x - 9B\sin 3x) + 9(A\cos 3x + B\sin 3x) = 0. \end{equation*}
    This gives zero, so the guess duplicates the homogeneous solution. Multiply by \(x\text{:}\)
    🔗
    \begin{equation*} y_p = x\bigl(A\cos 3x + B\sin 3x\bigr). \end{equation*}
    Compute \(y_p'' + 9y_p\text{.}\) After simplification (CAS recommended), one obtains:
    🔗
    \begin{equation*} y_p'' + 9y_p = 3A \sin 3x - 3B \cos 3x. \end{equation*}
    Match coefficients with the forcing \(\cos 3x + \sin 3x\text{:}\)
    🔗
    \begin{equation*} -3B = 1,\qquad 3A = 1. \end{equation*}
    So \(A = \tfrac13\text{,}\) \(B = -\tfrac13\text{.}\)
    🔗
    Thus
    🔗
    \begin{equation*} y_p = \frac{x}{3}\bigl(\cos 3x - \sin 3x\bigr). \end{equation*}
    Full solution:
    🔗
    \begin{equation*} y = C_1 \cos 3x + C_2 \sin 3x + \frac{x}{3}(\cos 3x - \sin 3x). \end{equation*}
    Apply initial conditions:
    🔗
    At \(x=0\text{:}\)
    🔗
    \begin{equation*} y(0) = C_1 = 2. \end{equation*}
    Derivative:
    🔗
    \begin{align*} y' \amp = -3C_1 \sin 3x + 3C_2 \cos 3x \\ \ \ \amp + \frac13(\cos 3x - \sin 3x) \\ \ \ \amp + \frac{x}{3}(-3\sin 3x - 3\cos 3x). \end{align*}
    At \(x=0\text{:}\)
    🔗
    \begin{equation*} y'(0) = 3C_2 + \frac13 = 1 \quad\Rightarrow\quad 3C_2 = \frac23 \quad\Rightarrow\quad C_2 = \frac{2}{9}. \end{equation*}
    Final solution:
    🔗
    \begin{equation*} y(x) = 2\cos 3x + \frac{2}{9}\sin 3x + \frac{x}{3}\bigl(\cos 3x - \sin 3x\bigr). \end{equation*}

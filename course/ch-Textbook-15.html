    Handout Daily Prep 3.2 - Transforms of Derivatives and ODEs
    

    Section Overview

    In this section, we learn how the Laplace transform interacts with derivatives and how this property becomes the key tool for solving ordinary differential equations in the \(s\)-domain. The text shows that differentiation in time becomes multiplication by \(s\) (with corrections coming from initial conditions), and provides a table of Laplace transforms for first, second, and third derivatives. Students then see how this framework converts an ODE into an algebraic equation, which can be solved for the transformed function and inverted to obtain the solution in the time domain.

    Exercises Exercises

    1. Laplace Transform of a First Derivative.
    What is the Laplace transform of \(y'(t)\text{?}\)
    🔗
    • \(sY(s) - y(0)\)
    • Correct. \({\cal L}\{y'\} = sY(s) - y(0)\text{,}\) which is crucial when solving ODEs with initial conditions.
    • \(sY(s)\)
    • Missing the important term \(-y(0)\text{.}\)
    • \(Y(s) - y(0)\)
    • Incorrect scaling—the factor \(s\) is essential.
    🔗
    2. Laplace Transform of a Second Derivative.
    What is \({\cal L}\{y''(t)\}\) in terms of \(Y(s)\text{,}\) \(y(0)\text{,}\) and \(y'(0)\text{?}\)
    🔗
    • \(s^{2}Y(s) - s\,y(0) - y'(0)\)
    • Correct. \({\cal L}\{y''\} = s^2Y(s) - s y(0) - y'(0)\text{,}\) used directly when turning ODEs into algebraic equations in \(s\text{.}\)
    • \(s^{2}Y(s) - y(0)\)
    • Missing the critical \(-s y(0)\) and \(-y'(0)\) terms.
    • \(sY(s) - y'(0)\)
    • This is the formula for first derivatives, not second.
    🔗
    3. Converting an ODE to an Algebraic Equation.
    Consider the IVP \(y'' + y = 0\) with \(y(0)=2\text{,}\) \(y'(0)=0\text{.}\) After taking Laplace transforms and using derivative rules, which algebraic equation in \(Y(s)\) results?
    🔗
    • \((s^{2}+1)Y(s) - 2s = 0\)
    • Correct. Using \({\cal L}\{y''\}=s^2Y - s y(0) - y'(0) = s^2Y -2s\) and \({\cal L}\{y\}=Y\text{.}\) So \((s^2+1)Y - 2s = 0\text{.}\)
    • \((s^{2}+1)Y(s)=0\)
    • Initial conditions were not incorporated.
    • \((s^{2}-1)Y(s)=2\)
    • Incorrect signs and missing \(s\)-term.
    🔗
    4. Solve for \(Y(s)\).
    From the algebraic equation \((s^{2}+1)Y(s) - 2s = 0\text{,}\) what is \(Y(s)\text{?}\)
    🔗
    • \(Y(s) = \dfrac{2s}{s^{2}+1}\)
    • Correct. Divide both sides by \(s^{2}+1\text{.}\) This corresponds to the transform of \(2\cos t\text{.}\)
    • \(Y(s)=\dfrac{2}{s^{2}+1}\)
    • Missing a factor of \(s\) in the numerator.
    • \(Y(s)=\dfrac{2s}{s^{2}-1}\)
    • Incorrect denominator.
    🔗
    5. Inverse Laplace Identification.
    Which of the following time‑domain functions have Laplace transforms equal to \(\dfrac{2s}{s^{2}+1}\text{?}\)
    🔗
    • \(2\cos t\)
    • Correct. Since \({\cal L}\{\cos t\} = s/(s^{2}+1)\text{,}\) multiplying by 2 gives the target transform.
    • \(2\sin t\)
    • No—its transform is \(2/(s^{2}+1)\text{,}\) not \(2s/(s^{2}+1)\text{.}\)
    • \(e^{-t}\)
    • No—its transform is \(1/(s+1)\text{.}\)
    🔗
    6.
    What is the next line in Table 3.2.1?
    🔗
    Solution.
    Table 3.2.1 lists transforms of derivatives. After the line
    🔗
    \begin{equation*} {\cal L}\{y'''(t)\} = s^3G(s) - s^2g(0)-sg'(0)-g''(0), \end{equation*}
    the next line is the transform of the second derivative:
    🔗
    \begin{equation*} {\cal L}\{y''''(t)\} = s^{4}G(s) - s^3g(0) - s^2g'(0)-sg''(0)-g'''(0). \end{equation*}

    Plot the following functions.
    🔗
    1. \(\displaystyle f(t) = u(t-\pi)\sin t\)
      🔗

    2. \(\displaystyle u(t-1) - u(t-2)\)
      🔗
      🔗

    🔗
    (c) This describes a “piecewise linear bump” between \(0\) and \(2\text{:}\)
    🔗
    \begin{equation*} f(t)= \begin{cases} t, \amp 0\le t < 1,\\ -t+2, \amp 1\le t < 2,\\ 0, \amp t\ge 2. \end{cases} \end{equation*}

    8.
    Using the Laplace transform, solve the IVP
    🔗
    \begin{equation*} y'' + 4y = \sin 3t, \quad y(0)=0,\; y'(0)=0. \end{equation*}
    Solution.
    Laplace transforming:
    🔗
    \begin{equation*} s^{2}Y(s) + 4Y(s) = \frac{3}{s^{2}+9}. \end{equation*}
    Solve for \(Y(s)\text{:}\)
    🔗
    \begin{equation*} Y(s) = \frac{3}{(s^{2}+4)(s^{2}+9)}. \end{equation*}
    Use partial fractions:
    🔗
    \begin{equation*} \frac{3}{(s^{2}+4)(s^{2}+9)} = \frac{3}{5}\frac{1}{s^{2}+4} - \frac{3}{5}\frac{1}{s^{2}+9}. \end{equation*}
    Invert term‑by‑term:
    🔗
    \begin{align*} y(t) \amp = \frac{3}{5}\cdot \frac12\sin 2t - \frac{3}{5}\cdot \frac13\sin 3t \\ \amp = \frac{3}{10}\sin 2t - \frac{1}{5}\sin 3t. \end{align*}
    🔗
    🔗
    9.
    Solve the IVP
    🔗
    \begin{equation*} y'' - y' - 6y = 0, \qquad y(0) = 2, \; y'(0) = -1. \end{equation*}
    using the Laplace transform.
    🔗
    Solution.
    Take Laplace transforms term-by-term:
    🔗
    \begin{equation*} \mathcal{L}\{y''\} = s^{2}Y(s) - s y(0) - y'(0) = s^{2}Y - 2s + 1, \end{equation*}
    \begin{equation*} \mathcal{L}\{y'\} = sY - y(0) = sY - 2, \end{equation*}
    \begin{equation*} \mathcal{L}\{y\} = Y. \end{equation*}
    Substitute into \(y'' - y' - 6y = 0\text{:}\)
    🔗
    \begin{equation*} (s^{2}Y - 2s + 1) - (sY - 2) - 6Y = 0. \end{equation*}
    Combine like terms:
    🔗
    \begin{equation*} (s^{2} - s - 6)Y - 2s + 3 = 0. \end{equation*}
    Solve for \(Y(s)\text{:}\)
    🔗
    \begin{equation*} Y(s) = \frac{2s - 3}{(s - 3)(s + 2)}. \end{equation*}
    Use partial fractions:
    🔗
    \begin{equation*} \frac{2s - 3}{(s - 3)(s + 2)} = \frac{3/5}{s - 3} + \frac{7/5}{s + 2}. \end{equation*}
    Invert term-by-term:
    🔗
    \begin{equation*} y(t) = \frac{3}{5} e^{3t} + \frac{7}{5} e^{-2t}. \end{equation*}
    This satisfies the differential equation and initial conditions, so it is the correct solution.
    🔗
    🔗
    🔗
    10.
    (Shifting Theorem) Suppose \({\cal L}[f(t)] = F(s)\) for \(s>c\text{.}\) Show that for any real \(a\) and \(s>a+c\text{,}\)
    🔗
    \begin{equation*} {\cal L}[e^{at} f(t)](s) = F(s-a). \end{equation*}
    Solution.
    This follows directly because
    🔗
    \begin{align*} {\cal L}[e^{at}f(t)](s) \amp = \int_0^\infty e^{-st} e^{at}f(t)\,dt \\ \amp = \int_0^\infty e^{-(s-a)t}f(t)\,dt \\ \amp = F(s-a). \end{align*}
    The domain shifts by replacing \(s\) with \(s-a\text{,}\) requiring \(s-a>c\text{.}\)
    🔗
    🔗
    🔗
    11.
    We know that \({\cal L}[t^{2}](s) = \frac{2}{s^{3}}\text{.}\) Compute \({\cal L}^{-1}\!\left[\frac{2}{(s-2)^{3}}\right]\text{.}\)
    🔗
    Solution.
    We start from the known transform
    🔗
    \begin{equation*} {\cal L}[t^{2}](s) = \frac{2}{s^{3}}. \end{equation*}
    The expression \(\dfrac{2}{(s-2)^{3}}\) is obtained by replacing \(s\) with \(s-2\) in \(\dfrac{2}{s^{3}}\text{.}\) By the (exponential) shifting theorem,
    🔗
    \begin{equation*} {\cal L}[e^{at}f(t)] = F(s-a). \end{equation*}
    Here, \(F(s) = \dfrac{2}{s^{3}}\) corresponds to \(f(t)=t^{2}\text{,}\) and we need \(F(s-2)\text{.}\) Therefore
    🔗
    \begin{equation*} {\cal L}^{-1}\!\left[\frac{2}{(s-2)^{3}}\right] = e^{2t} \, t^{2}. \end{equation*}
    Thus the inverse Laplace transform is
    🔗
    \begin{equation*} {\cal L}^{-1}\!\left[\frac{2}{(s-2)^{3}}\right] = t^{2} e^{2t}. \end{equation*}
    🔗
    🔗
    12.
    Define
    🔗
    \begin{equation*} f(t) = \left\{ \begin{array}{cr} 1 \amp 0 \le t < 1 \\ 3 \amp 1 \le t < 2 \\ t \amp 2 \le t < \infty \end{array} \right\}. \end{equation*}
    Compute the Laplace transform \({\cal L}[f(t)]\text{.}\) Hint: Write
    🔗
    \begin{equation*} f(t) = \bigl[1 - 1\cdot u(t-1)\bigr] + \bigl[3u(t-1) - 3u(t-2)\bigr] + \bigl[t\,u(t-2)\bigr]. \end{equation*}
    Solution.
    Using the decomposition:
    🔗
    1. First term:
      🔗
      \begin{equation*} {\cal L}[\,1 - u(t-1)\,](s) = \frac{1}{s} - \frac{e^{-s}}{s}. \end{equation*}
      🔗
    2. Second term:
      🔗
      \begin{equation*} {\cal L}[\,3u(t-1) - 3u(t-2)\,](s) = \frac{3e^{-s}}{s} - \frac{3e^{-2s}}{s}. \end{equation*}
      🔗
    3. Third term: We use the shifting rule \({\cal L}[\,t\,u(t-a)\,] = e^{-as}\Bigl(\frac{1}{s^{2}} + \frac{a}{s}\Bigr)\text{.}\)
      🔗
      Here \(a=2\text{,}\) so
      🔗
      \begin{equation*} {\cal L}[\,t\,u(t-2)\,](s) = e^{-2s}\!\left(\frac{1}{s^{2}} + \frac{2}{s}\right). \end{equation*}
      🔗
    Now sum all terms:
    🔗
    \begin{equation*} {\cal L}[f(t)](s) = \left(\frac{1}{s} - \frac{e^{-s}}{s}\right) + \left(\frac{3e^{-s}}{s} - \frac{3e^{-2s}}{s}\right) + e^{-2s}\!\left(\frac{1}{s^{2}} + \frac{2}{s}\right). \end{equation*}
    Combine like terms:
    🔗
    \begin{equation*} {\cal L}[f(t)] = \frac{1}{s} + \frac{2e^{-s}}{s} + e^{-2s}\!\left( \frac{1}{s^{2}} + \frac{2}{s} - \frac{3}{s} \right). \end{equation*}
    Simplify the expression inside the final brackets:
    🔗
    \begin{equation*} \frac{1}{s^{2}} + \frac{2}{s} - \frac{3}{s} = \frac{1}{s^{2}} - \frac{1}{s}. \end{equation*}
    Thus the Laplace transform is
    🔗
    \begin{equation*} {\cal L}[f(t)](s) = \frac{1}{s} + \frac{2e^{-s}}{s} + e^{-2s}\!\left(\frac{1}{s^{2}} - \frac{1}{s}\right). \end{equation*}
    🔗
    🔗
    13.
    A mass that weighs 32 lb (\(m=1\) slug) is attached to a spring that is stretched 1 ft by a force of 4 lb (\(k=4\) lb/ft). The mass is initially at rest in equilibrium. A forcing function \(f(t)=1\) is applied for \(0 \le t < \pi\) and then turned off for \(t\ge \pi\text{.}\) (Assume no damping.) Find the resulting position \(x(t)\text{.}\)
    🔗
    Solution.
    1. Model the equation.
    🔗
    No damping → equation is
    🔗
    \begin{equation*} x'' + 4x = f(t). \end{equation*}
    Since the forcing is 1 for \(0\le t < \pi\) and 0 afterward,
    🔗
    \begin{equation*} f(t) = 1 - u(t-\pi). \end{equation*}
    Initial conditions: \(x(0)=0\text{,}\) \(x'(0)=0\text{.}\)
    🔗
    2. Laplace transform.
    🔗
    Transform the ODE:
    🔗
    \begin{equation*} s^{2}X(s) + 4X(s) = \frac{1}{s} - \frac{e^{-\pi s}}{s}. \end{equation*}
    Solve:
    🔗
    \begin{equation*} X(s) = \frac{1}{s(s^{2}+4)} - \frac{e^{-\pi s}}{s(s^{2}+4)}. \end{equation*}
    3. Inverse transform of the unshifted term.
    🔗
    Use the known identity:
    🔗
    \begin{equation*} \mathcal{L}^{-1}\!\left[\frac{1}{s(s^{2}+a^{2})}\right] = \frac{1}{a^{2}}\bigl(1 - \cos(at)\bigr). \end{equation*}
    Here \(a=2\text{,}\) so
    🔗
    \begin{equation*} \mathcal{L}^{-1}\!\left[\frac{1}{s(s^{2}+4)}\right] = \frac{1}{4}\bigl(1 - \cos(2t)\bigr). \end{equation*}
    4. Inverse transform of the shifted piece.
    🔗
    Use the shifting theorem:
    🔗
    \begin{equation*} \mathcal{L}^{-1}\!\left[ e^{-a s} F(s) \right] = u(t-a)\, f(t-a), \end{equation*}
    where \(f(t)=\mathcal{L}^{-1}[F(s)]\text{.}\) In our case:
    🔗
    \begin{equation*} F(s)=\frac{1}{s(s^{2}+4)},\qquad f(t)=\frac{1}{4}\bigl(1 - \cos(2t)\bigr). \end{equation*}
    Thus:
    🔗
    \begin{equation*} \mathcal{L}^{-1}\!\left[ \frac{e^{-\pi s}}{s(s^{2}+4)} \right] = u(t-\pi)\, \frac{1}{4} \bigl(1 - \cos(2(t-\pi))\bigr). \end{equation*}
    5. Final solution.
    🔗
    Combining the pieces:
    🔗
    \begin{equation*} x(t) = \frac14\bigl(1 - \cos(2t)\bigr) \;-\; u(t-\pi) \frac14\bigl(1 - \cos(2(t-\pi))\bigr). \end{equation*}
    Since \(\cos(2(t-\pi))=\cos(2t)\text{,}\) the expression simplifies nicely:
    🔗
    \begin{equation*} x(t) = \frac14\bigl(1 - \cos(2t)\bigr) - u(t-\pi)\frac14\bigl(1 - \cos(2t)\bigr). \end{equation*}
    Factor:
    🔗
    \begin{equation*} x(t) = \frac14\bigl(1 - \cos(2t)\bigr)\, \bigl[1 - u(t-\pi)\bigr]. \end{equation*}
    In piecewise form:
    🔗
    \begin{equation*} x(t) = \begin{cases} \frac14\bigl(1 - \cos(2t)\bigr), \amp 0 \le t < \pi,\\[6pt] 0, \amp t \ge \pi. \end{cases} \end{equation*}
    Physically: the system oscillates between 0 and \(\pi\text{,}\) then returns to equilibrium once the forcing stops.

    14.
    Use the definition of the Laplace transform to compute (by hand) \({\cal L}[f(t)]\) for the function \(f\) graphed in Figure 18. Then use technology to verify your result.
    🔗
    Solution.
    From the graph, the function is piecewise-defined:
    🔗
    \begin{equation*} f(t) = \begin{cases} 0 \amp 0 \le t < 1,\\[4pt] t-1 \amp t \ge 1. \end{cases} \end{equation*}
    Therefore the Laplace transform is
    🔗
    \begin{equation*} {\cal L}[f(t)](s) = \int_{0}^{1} 0 \cdot e^{-st}\, dt + \int_{1}^{\infty} (t-1)e^{-st}\, dt = \int_{1}^{\infty} (t-1)e^{-st}\, dt. \end{equation*}
    Compute the integral.
    🔗
    Substitute \(u=t-1\text{,}\) hence \(t=u+1\) and when \(t=1\text{,}\) \(u=0\text{.}\) Then
    🔗
    \begin{equation*} \int_{1}^{\infty} (t-1)e^{-st}\, dt = \int_{0}^{\infty} u\, e^{-s(u+1)}\, du = e^{-s} \int_{0}^{\infty} u\, e^{-su}\, du. \end{equation*}
    Compute the remaining integral:
    🔗
    \begin{equation*} \int_{0}^{\infty} u\, e^{-su}\, du = \frac{1}{s^{2}}. \end{equation*}
    Hence
    🔗
    \begin{equation*} {\cal L}[f(t)](s) = e^{-s}\, \frac{1}{s^{2}} = \frac{e^{-s}}{s^{2}}. \end{equation*}
    🔗
    🔗
    15.
    Use partial fractions to evaluate
    🔗
    \begin{equation*} {\cal L}^{-1}\!\left\{ \frac{s^{2} + 6s + 9}{(s-1)(s-2)(s+4)} \right\} \end{equation*}
    by hand. Use technology to verify the result.
    🔗
    Solution.
    1. Set up partial fractions.
    🔗
    We want to write
    🔗
    \begin{equation*} \frac{s^{2}+6s+9}{(s-1)(s-2)(s+4)} = \frac{A}{s-1} + \frac{B}{s-2} + \frac{C}{s+4}. \end{equation*}
    Multiply through:
    🔗
    \begin{equation*} s^{2}+6s+9 = A(s-2)(s+4) + B(s-1)(s+4) + C(s-1)(s-2). \end{equation*}
    2. Evaluate at convenient values.
    🔗
    • \(s=1\text{:}\)
      🔗
      \begin{equation*} 1+6+9 = A(1-2)(1+4) \quad\Longrightarrow\quad 16 = -5A \quad\Longrightarrow\quad A = -\frac{16}{5}. \end{equation*}
      🔗
    • \(s=2\text{:}\)
      🔗
      \begin{equation*} 4 + 12 + 9 = B(2-1)(2+4) \quad\Longrightarrow\quad 25 = 6B \quad\Longrightarrow\quad B = \frac{25}{6}. \end{equation*}
      🔗
    • \(s=-4\text{:}\)
      🔗
      \begin{equation*} 16 - 24 + 9 = C(-4-1)(-4-2) \\[4pt] 1 = C( -5)( -6 ) \quad\Longrightarrow\quad C = \frac{1}{30}. \end{equation*}
      🔗
    3. Write the decomposition.
    🔗
    \begin{equation*} \frac{s^{2}+6s+9}{(s-1)(s-2)(s+4)} = -\frac{16}{5}\frac{1}{s-1} + \frac{25}{6}\frac{1}{s-2} + \frac{1}{30}\frac{1}{s+4}. \end{equation*}
    4. Inverse Laplace transform term‑by‑term.
    🔗
    Use \({\cal L}^{-1}\{1/(s-a)\} = e^{at}\text{.}\) Thus:
    🔗
    \begin{equation*} y(t) = -\frac{16}{5} e^{t} + \frac{25}{6} e^{2t} + \frac{1}{30} e^{-4t}. \end{equation*}
    🔗
    🔗
    16.
    Solve the IVP
    🔗
    \begin{equation*} y'' + 2y' + y = f(t), \qquad y(0)=0,\; y'(0)=0, \end{equation*}
    where
    🔗
    \begin{equation*} f(t) = \left\{ \begin{array}{cr} 0 \amp 0 \le t < 1 \\[4pt] 1 \amp 1 \le t < 2 \\[4pt] -1 \amp 2 \le t < 3 \\[4pt] 0 \amp 3 \le t < \infty \end{array} \right\}. \end{equation*}
    Solution.
    1. Rewrite the forcing using step functions.
    🔗
    The piecewise function becomes:
    🔗
    \begin{equation*} f(t) = u(t-1) - 2u(t-2) + u(t-3). \end{equation*}
    (Check: On \([1,2)\) it equals \(1\text{;}\) on \([2,3)\) it equals \(-1\text{;}\) for \(t\ge 3\) it returns to \(0\text{.}\))
    🔗
    2. Apply the Laplace transform.
    🔗
    With \(y(0)=0\text{,}\) \(y'(0)=0\text{:}\)
    🔗
    \begin{equation*} \mathcal{L}\{y''\} = s^{2}Y(s),\qquad \mathcal{L}\{2y'\} = 2sY(s),\qquad \mathcal{L}\{y\} = Y(s). \end{equation*}
    So the ODE becomes:
    🔗
    \begin{equation*} (s^{2} + 2s + 1) Y(s) = \frac{e^{-s}}{s} - \frac{2e^{-2s}}{s} + \frac{e^{-3s}}{s}. \end{equation*}
    Since \(s^{2} + 2s + 1 = (s+1)^{2}\text{,}\) we have:
    🔗
    \begin{equation*} Y(s) = \frac{e^{-s}}{s(s+1)^{2}} - \frac{2e^{-2s}}{s(s+1)^{2}} + \frac{e^{-3s}}{s(s+1)^{2}}. \end{equation*}
    3. Invert the non‑shifted kernel.
    🔗
    First compute
    🔗
    \begin{equation*} h(t) = \mathcal{L}^{-1}\!\left[\frac{1}{s(s+1)^{2}}\right]. \end{equation*}
    Do partial fractions:
    🔗
    \begin{equation*} \frac{1}{s(s+1)^{2}} = \frac{1}{s} - \frac{1}{s+1} - \frac{1}{(s+1)^{2}}. \end{equation*}
    Thus:
    🔗
    \begin{equation*} h(t) = 1 - e^{-t} - t e^{-t}. \end{equation*}
    4. Apply the shifting theorem.
    🔗
    Using
    🔗
    \begin{equation*} \mathcal{L}^{-1}\!\left[\frac{e^{-as}}{s(s+1)^{2}}\right] = u(t-a)\,h(t-a), \end{equation*}
    the solution becomes:
    🔗
    \begin{equation*} y(t) = u(t-1)h(t-1) - 2u(t-2)h(t-2) + u(t-3)h(t-3), \end{equation*}
    with \(h(t) = 1 - e^{-t} - t e^{-t}\text{.}\)
    🔗
    5. Final explicit form.
    🔗
    \begin{align*} y(t) \amp = u(t-1)\!\left[\,1 - e^{-(t-1)} - (t-1)e^{-(t-1)}\right] \\ \ \ \amp - 2u(t-2)\!\left[\,1 - e^{-(t-2)} - (t-2)e^{-(t-2)}\right] \\ \ \ \amp + u(t-3)\!\left[\,1 - e^{-(t-3)} - (t-3)e^{-(t-3)}\right]. \end{align*}

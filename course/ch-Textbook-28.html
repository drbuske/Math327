    Handout Daily Prep 5.7 - Multiple Eigenvalues
    

    Section Overview

    We now examine what happens in linear systems of ODEs when the coefficient matrix possesses multiple eigenvalues, especially repeated real eigenvalues. The section explains that having a repeated eigenvalue \(\lambda\) does not guarantee that the system has enough linearly independent eigenvectors to form a complete solution set. When only one eigenvector exists for a repeated eigenvalue, the system is called defective, and a generalized eigenvector must be found to construct a second linearly independent solution of the form \((\vec{v} t + \vec{w}) e^{\lambda t}\text{.}\) The method parallels the procedure for repeated roots of constantâ€‘coefficient scalar equations but must be handled in vector form. The section shows how generalized eigenvectors arise from solving \((A - \lambda I) \vec{w} = \vec{v}\text{,}\) and illustrates how the resulting solutions produce phaseâ€‘plane behavior distinct from the simple node cases with two eigenvectors. These ideas are essential for completing the eigenvalue method when the matrix is not diagonalizable.

    Exercises Exercises

    1. Recognizing a Repeated Eigenvalue.
    The matrix \(A = \begin{bmatrix} 3 \amp 1 \\ 0 \amp 3 \end{bmatrix}\) has which eigenvalues?
    ðŸ”—
    • A repeated eigenvalue \(\lambda = 3\text{.}\)
    • Correct. The characteristic polynomial is \((3-\lambda)^2\text{,}\) so the eigenvalue \(3\) has multiplicity 2.
    • Eigenvalues \(3\) and \(1\text{.}\)
    • Noâ€”the diagonal entries do not directly give distinct eigenvalues in this case.
    • Eigenvalues \(1\) and \(3\) with multiplicity one each.
    • Noâ€”the eigenvalue is repeated, not distinct.
    ðŸ”—
    2. Determining When a Matrix Is Defective.
    A matrix with a repeated eigenvalue \(\lambda\) is called defective when:
    ðŸ”—
    • It has fewer independent eigenvectors than the multiplicity of \(\lambda\text{.}\)
    • Correct. A repeated eigenvalue requires enough eigenvectors; if only one exists, the matrix is defective.
    • Its characteristic polynomial cannot be factored.
    • No. This has nothing to do with being defective.
    • Its trace is zero.
    • No. The trace does not determine diagonalizability.
    ðŸ”—
    3. Purpose of a Generalized Eigenvector.
    If a matrix has only one eigenvector \(\vec{v}\) for a repeated eigenvalue, why do we introduce a generalized eigenvector \(\vec{w}\text{?}\)
    ðŸ”—
    • To construct a second independent solution of the form \((t\vec{v} + \vec{w}) e^{\lambda t}\text{.}\)
    • Correct. A generalized eigenvector provides the missing second solution when only one eigenvector exists.
    • To find a new eigenvalue.
    • Noâ€”generalized eigenvectors do not create new eigenvalues.
    • To force the matrix to become diagonalizable.
    • No. A defective matrix cannot be made diagonalizable by choosing more vectors.
    ðŸ”—
    4. Equation for Finding a Generalized Eigenvector.
    If \(\vec{v}\) is an eigenvector for a repeated eigenvalue \(\lambda\text{,}\) which equation must a generalized eigenvector \(\vec{w}\) satisfy?
    ðŸ”—
    • \((A - \lambda I)\vec{w} = \vec{v}\)
    • Correct. This equation produces a vector \(\vec{w}\) that generates the second independent solution.
    • \(A\vec{w} = \lambda\vec{w}\)
    • Noâ€”that would make \(\vec{w}\) another eigenvector, which does not exist.
    • \((A - \lambda I)^2 \vec{w} = 0\)
    • Not the defining equationâ€”the generalized eigenvector is found using \((A-\lambda I)\vec{w} = \vec{v}\text{.}\)
    ðŸ”—
    5. Structure of the General Solution for a Defective Matrix.
    If a matrix has a repeated eigenvalue \(\lambda\) with only one eigenvector \(\vec{v}\) and a generalized eigenvector \(\vec{w}\text{,}\) what does the general solution to \(\vec{x}' = A\vec{x}\) look like?
    ðŸ”—
    • \(\vec{x}(t) = C_1 \vec{v} e^{\lambda t} + C_2 (\vec{v} t + \vec{w}) e^{\lambda t}\)
    • Correct. This is the full solution when only one eigenvector exists.
    • \(\vec{x}(t) = C_1 \vec{v} e^{\lambda t} + C_2 \vec{w} e^{\lambda t}\)
    • Noâ€”this does not produce two independent solutions.
    • \(\vec{x}(t) = (C_1 + C_2)t\, e^{\lambda t}\)
    • Noâ€”solutions must be vectorâ€‘valued and use \(\vec{v}\) and \(\vec{w}\text{.}\)

    6. Checking Whether a Matrix Is Defective.
    Consider the matrix \(A = \begin{bmatrix} 4 \amp 2 \\ 0 \amp 4 \end{bmatrix}.\)
    ðŸ”—
    1. Compute the eigenvalue(s) of \(A\text{.}\)
      ðŸ”—
      ðŸ”—
    2. Find all eigenvectors of \(A\) and determine whether the matrix is defective.
      ðŸ”—
      ðŸ”—
    Solution.
    (a) The characteristic polynomial is \(\det(A - \lambda I) = (4-\lambda)^2\text{,}\) so the only eigenvalue is the repeated eigenvalue \(\lambda = 4\text{.}\)
    ðŸ”—
    (b) To find the eigenvectors, solve \((A - 4I)\vec{v} = 0\text{.}\) That matrix is
    ðŸ”—
    \begin{equation*} A - 4I = \begin{bmatrix} 0 \amp 2 \\ 0 \amp 0 \end{bmatrix}. \end{equation*}
    The equation \(2v_2 = 0\) implies \(v_2 = 0\text{,}\) while \(v_1\) is free. Thus every eigenvector has the form
    ðŸ”—
    \begin{equation*} \vec{v} = \begin{bmatrix} 1 \\ 0 \end{bmatrix} c. \end{equation*}
    There is only one linearly independent eigenvector, even though the eigenvalue has algebraic multiplicity 2. Hence \(A\) is defective.

    7. Finding a Generalized Eigenvector.
    Let \(A = \begin{bmatrix} 5 \amp 1 \\ 0 \amp 5 \end{bmatrix},\) which has a repeated eigenvalue \(\lambda = 5\text{.}\) The unique eigenvector (up to scaling) is \(\vec{v} = \begin{bmatrix} 1 \\ 0 \end{bmatrix}\text{.}\)
    ðŸ”—
    Find a generalized eigenvector \(\vec{w}\) satisfying \((A - 5I)\vec{w} = \vec{v}\text{.}\)
    ðŸ”—
    Solution.
    Compute
    ðŸ”—
    \begin{equation*} A - 5I = \begin{bmatrix} 0 \amp 1 \\ 0 \amp 0 \end{bmatrix}. \end{equation*}
    We seek \(\vec{w} = \begin{bmatrix} w_1 \\ w_2 \end{bmatrix}\) such that
    ðŸ”—
    \begin{equation*} (A - 5I)\vec{w} = \begin{bmatrix} w_2 \\ 0 \end{bmatrix} = \begin{bmatrix} 1 \\ 0 \end{bmatrix}. \end{equation*}
    Thus \(w_2 = 1\text{,}\) and \(w_1\) is arbitrary. The simplest choice is
    ðŸ”—
    \begin{equation*} \vec{w} = \begin{bmatrix} 0 \\ 1 \end{bmatrix}. \end{equation*}
    This exactly matches the definition of a generalized eigenvector: solve \((A - \lambda I)\vec{w} = \vec{v}\text{.}\)
    ðŸ”—
    ðŸ”—
    ðŸ”—
    8. Constructing the Full Solution for a Defective Matrix.
    For the matrix \(A = \begin{bmatrix} 2 \amp 1 \\ 0 \amp 2 \end{bmatrix},\) the eigenvalue is \(\lambda = 2\) with eigenvector \(\vec{v} = \begin{bmatrix}1 \\ 0\end{bmatrix}\text{.}\) A generalized eigenvector is \(\vec{w} = \begin{bmatrix}0 \\ 1\end{bmatrix}\text{.}\)
    ðŸ”—
    Write the general solution to \(\vec{x}' = A\vec{x}\text{.}\)
    ðŸ”—
    Solution.
    A defective matrix with one eigenvector and one generalized eigenvector yields solutions
    ðŸ”—
    \begin{equation*} \vec{x}_1(t) = \vec{v} e^{2t}, \vec{x}_2(t) = (\vec{v} t + \vec{w}) e^{2t}. \end{equation*}
    Therefore the general solution is
    ðŸ”—
    \begin{equation*} \vec{x}(t) = C_1 \begin{bmatrix} 1 \\ 0 \end{bmatrix} e^{2t} + C_2 \left( t \begin{bmatrix} 1 \\ 0 \end{bmatrix} + \begin{bmatrix} 0 \\ 1 \end{bmatrix} \right) e^{2t}. \end{equation*}
    This is the standard repeatedâ€‘eigenvalue solution for a defective system.

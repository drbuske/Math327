    Handout Daily Prep 5.8 - Matrix Exponentials
    

    Section Overview

    It is time to introduce the matrix exponential -- a powerful method for solving linear constantâ€‘coefficient systems of differential equations of the form \(\vec{x}' = P \vec{x}\text{.}\) Motivated by the scalar solution \(x = e^{pt}\text{,}\) the section defines the matrix exponential \(e^{tP}\) via its convergent Taylor series \(I + tP + \tfrac{1}{2}t^2 P^2 + \cdots\text{,}\) and demonstrates that this function satisfies the key identity \(\tfrac{d}{dt}(e^{tP}) = P e^{tP}\text{.}\) This establishes \(\vec{x}(t) = e^{tP} \vec{c}\) as the general solution to the system, and in particular \(\vec{x}(0)=\vec{c}\text{.}\) The section explains that matrix exponentials provide a fundamental matrix solution and give a clean way to solve initial value problems directly. It also highlights an important subtlety: in contrast to real numbers, matrices do not generally commute, so \(e^{A+B} \neq e^{A} e^{B}\) unless \(A\) and \(B\) satisfy \(AB = BA\text{.}\) Finally, the section explores simple casesâ€”especially diagonal and triangular matricesâ€”where matrix exponentials can be computed efficiently.

    Which of the following correctly defines the matrix exponential \(e^{tP}\) for a square matrix \(P\text{?}\)
    ðŸ”—

    • \(e^{tP} = (I - tP)^{-1}\)
    • Noâ€”this only works for special matrices, not in general.
    • \(e^{tP} = P^t\)

    2. Why \(e^{tP}\) Solves the System.
    Why does \(\vec{x}(t) = e^{tP} \vec{c}\) solve the system \(\vec{x}' = P\vec{x}\text{?}\)
    ðŸ”—
    • Because \(\frac{d}{dt}(e^{tP}) = P e^{tP}\text{.}\)
    • Correct. Differentiating the series termâ€‘byâ€‘term gives \(\frac{d}{dt}(e^{tP}) = P e^{tP}\text{,}\) so multiplying by \(\vec{c}\) yields a solution.
    • Because \(e^{tP}\) diagonalizes every matrix \(P\text{.}\)
    • Noâ€”only some matrices are diagonalizable.
    • Because \(e^{tP}\vec{c} = \vec{c}\) for all \(t\text{.}\)
    • Noâ€”this is only true when \(P = 0\text{.}\)
    ðŸ”—
    3. Solving an IVP with the Matrix Exponential.
    For the system \(\vec{x}' = P\vec{x}\) with initial condition \(\vec{x}(0) = \vec{b}\text{,}\) what is the solution?
    ðŸ”—
    • \(\vec{x}(t) = e^{tP}\vec{b}\)
    • Correct. The general solution is \(\vec{x}(t)=e^{tP}\vec{c}\text{,}\) and \(\vec{x}(0)=\vec{b}\) implies \(\vec{c} = \vec{b}\text{.}\)
    • \(\vec{x}(t) = P^t \vec{b}\)
    • Noâ€”this expression is not defined and is not the solution.
    • \(\vec{x}(t) = e^{t} P \vec{b}\)
    • Noâ€”the exponential must apply to the matrix, not to a scalar.
    ðŸ”—
    4. Matrix Exponential of a Diagonal Matrix.
    If \(P = \begin{bmatrix} 2 \amp 0 \\ 0 \amp -1 \end{bmatrix},\) what is \(e^{tP}\text{?}\)
    ðŸ”—
    • \(e^{tP} = \begin{bmatrix} e^{2t} \amp 0 \\ 0 \amp e^{-t} \end{bmatrix}\)
    • Correct. For diagonal matrices, the exponential is taken on each diagonal entry.
    • \(e^{tP} = e^{t} P\)
    • Noâ€”the exponential does not distribute in that way.
    • \(e^{tP} = \begin{bmatrix} 2e^{t} \amp 0 \\ 0 \amp -e^{t} \end{bmatrix}\)
    • Noâ€”the exponentials must be applied directly to the diagonal entries.
    ðŸ”—
    5. When Does \(e^{A+B} = e^{A} e^{B}\) Hold?
    When is the identity \(e^{A+B} = e^{A} e^{B}\) true?
    ðŸ”—
    • When \(A\) and \(B\) commute: \(AB = BA\text{.}\)
    • Correct. The section warns that the identity generally fails unless \(A\) and \(B\) commute.
    • Always, for all square matrices.
    • Noâ€”this is false even for simple 2Ã—2 matrices.
    • Only when both \(A\) and \(B\) are diagonal.
    • Not quiteâ€”diagonal matrices commute, but commutativity (not diagonality) is the real requirement.
    ðŸ”—
    6. Using the Series Definition to Compute a Simple Matrix Exponential.
    Let \(P = \begin{bmatrix} 0 \amp 1 \\ 0 \amp 0 \end{bmatrix}.\) Compute \(e^{tP}\) directly from the power series definition of the matrix exponential: \(e^{tP} = I + tP + \frac{t^2}{2!} P^2 + \frac{t^3}{3!} P^3 + \cdots.\)
    ðŸ”—
    Solution.
    First compute powers of \(P\text{.}\) Since \(P = \begin{bmatrix} 0 \amp 1 \\ 0 \amp 0 \end{bmatrix},\) we find:
    ðŸ”—
    \begin{equation*} P^2 = \begin{bmatrix} 0 \amp 0 \\ 0 \amp 0 \end{bmatrix}, P^3 = 0, \qquad \text{and all higher powers are also zero.} \end{equation*}
    Thus the power series truncates after the first term involving \(P\text{:}\)
    ðŸ”—
    \begin{equation*} e^{tP} = I + tP = \begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix} + t \begin{bmatrix} 0 \amp 1 \\ 0 \amp 0 \end{bmatrix} = \begin{bmatrix} 1 \amp t \\ 0 \amp 1 \end{bmatrix}. \end{equation*}
    Nilpotent matrices make the exponential easy to compute using the series.

    ðŸ”—

    7. Solving an Initial Value Problem Using the Matrix Exponential.
    Consider the system \(\vec{x}' = \begin{bmatrix} 2 \amp 0 \\ 0 \amp -3 \end{bmatrix} \vec{x}, \vec{x}(0) = \begin{bmatrix} 5 \\ -2 \end{bmatrix}.\) Use the matrix exponential to solve for \(\vec{x}(t)\text{.}\)
    ðŸ”—
    Solution.
    Because the matrix is diagonal, the matrix exponential is computed by exponentiating each diagonal entry:
    ðŸ”—
    \begin{equation*} e^{tP} = e^{t \begin{bmatrix} 2 \amp 0 \\ 0 \amp -3 \end{bmatrix}} = \begin{bmatrix} e^{2t} \amp 0 \\ 0 \amp e^{-3t} \end{bmatrix}. \end{equation*}
    The solution to \(\vec{x}' = P\vec{x}\) with \(\vec{x}(0) = \vec{b}\) is \(\vec{x}(t) = e^{tP}\vec{b}\text{.}\) Therefore:
    ðŸ”—
    \begin{equation*} \vec{x}(t) = \begin{bmatrix} e^{2t} \amp 0 \\ 0 \amp e^{-3t} \end{bmatrix} \begin{bmatrix} 5 \\ -2 \end{bmatrix} = \begin{bmatrix} 5 e^{2t} \\ -2 e^{-3t} \end{bmatrix}. \end{equation*}
    ðŸ”—
    ðŸ”—
    8. Understanding Commutativity and Matrix Exponential Identities.
    Let \(A = \begin{bmatrix} 1 \amp 0 \\ 0 \amp 2 \end{bmatrix}, B = \begin{bmatrix} 3 \amp 0 \\ 0 \amp -1 \end{bmatrix}.\)
    ðŸ”—
    1. Show that \(A\) and \(B\) commute.
      ðŸ”—
      ðŸ”—
    2. Use this fact to compute \(e^{A+B}\) in terms of \(e^{A}\) and \(e^{B}\text{.}\)
      ðŸ”—
      ðŸ”—
    Solution.
    (a) Compute \(AB\) and \(BA\text{:}\)
    ðŸ”—
    \begin{equation*} AB = \begin{bmatrix} 1 \amp 0 \\ 0 \amp 2 \end{bmatrix} \begin{bmatrix} 3 \amp 0 \\ 0 \amp -1 \end{bmatrix} = \begin{bmatrix} 3 \amp 0 \\ 0 \amp -2 \end{bmatrix}, \end{equation*}
    \begin{equation*} BA = \begin{bmatrix} 3 \amp 0 \\ 0 \amp -1 \end{bmatrix} \begin{bmatrix} 1 \amp 0 \\ 0 \amp 2 \end{bmatrix} = \begin{bmatrix} 3 \amp 0 \\ 0 \amp -2 \end{bmatrix}. \end{equation*}
    Thus \(AB = BA\text{.}\) When matrices commute, the exponential identity \(e^{A+B} = e^{A} e^{B}\) holds.
    ðŸ”—
    (b) Since \(A+B\) is diagonal,
    ðŸ”—
    \begin{equation*} A+B = \begin{bmatrix} 4 \amp 0 \\ 0 \amp 1 \end{bmatrix}, e^{A+B} = \begin{bmatrix} e^{4} \amp 0 \\ 0 \amp e^{1} \end{bmatrix}. \end{equation*}
    Also,
    ðŸ”—
    \begin{equation*} e^{A} = \begin{bmatrix} e^{1} \amp 0 \\ 0 \amp e^{2} \end{bmatrix}, e^{B} = \begin{bmatrix} e^{3} \amp 0 \\ 0 \amp e^{-1} \end{bmatrix}. \end{equation*}
    Multiply them:
    ðŸ”—
    \begin{equation*} e^{A} e^{B} = \begin{bmatrix} e^{4} \amp 0 \\ 0 \amp e^{1} \end{bmatrix} = e^{A+B}. \end{equation*}
    This confirms the commutativityâ€‘dependent identity emphasized in the section.

    ðŸ”—
    ðŸ”—
    9. Matrix Exponential of a Diagonal 3Ã—3 Matrix.
    Compute the matrix exponential \(e^{tA}\) for the diagonal matrix \(A = \begin{bmatrix} -2 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 4 \end{bmatrix}.\)
    ðŸ”—
    Solution.
    For a diagonal matrix, the matrix exponential is obtained by exponentiating each diagonal entry individually.
    ðŸ”—
    \begin{equation*} e^{tA} = \begin{bmatrix} e^{-2t} \amp 0 \amp 0 \\ 0 \amp e^{t} \amp 0 \\ 0 \amp 0 \amp e^{4t} \end{bmatrix}. \end{equation*}
    No offâ€‘diagonal terms appear since powers of a diagonal matrix remain diagonal and the Taylor series keeps all entries separated.

    ðŸ”—

    10. Solving a 3Ã—3 System Using the Matrix Exponential.
    Consider the system \(\vec{x}' = A \vec{x}, \qquad A = \begin{bmatrix} 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 0 \end{bmatrix},\) with initial condition \(\vec{x}(0) = \begin{bmatrix} 3 \\ 0 \\ 2 \end{bmatrix}.\) Compute \(e^{tA}\) and then find \(\vec{x}(t)\text{.}\)
    ðŸ”—
    Solution.
    First compute powers of \(A\text{.}\) This matrix is strictly upper triangular, so it is nilpotent:
    ðŸ”—
    \begin{equation*} A^2 = \begin{bmatrix} 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \end{bmatrix}, A^3 = 0, A^k = 0 \text{ for all } k \ge 3. \end{equation*}
    By the series definition of the matrix exponential \(e^{tA} = I + tA + \frac{t^2}{2}A^2,\) since all higher powers vanish.
    ðŸ”—
    \begin{equation*} e^{tA} = \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix} + t \begin{bmatrix} 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 0 \end{bmatrix} + \frac{t^2}{2} \begin{bmatrix} 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \end{bmatrix}. \end{equation*}
    Adding these gives
    ðŸ”—
    \begin{equation*} e^{tA} = \begin{bmatrix} 1 \amp t \amp \tfrac{1}{2}t^2 \\ 0 \amp 1 \amp t \\ 0 \amp 0 \amp 1 \end{bmatrix}. \end{equation*}
    Now compute \(\vec{x}(t) = e^{tA} \vec{x}(0)\text{:}\)
    ðŸ”—
    \begin{equation*} \vec{x}(t) = \begin{bmatrix} 1 \amp t \amp \tfrac12 t^2 \\ 0 \amp 1 \amp t \\ 0 \amp 0 \amp 1 \end{bmatrix} \begin{bmatrix} 3 \\ 0 \\ 2 \end{bmatrix} = \begin{bmatrix} 3 + \tfrac12 t^2 \cdot 2 \\ 0 + t \cdot 2 \\ 2 \end{bmatrix} = \begin{bmatrix} 3 + t^2 \\ 2t \\ 2 \end{bmatrix}. \end{equation*}
    Therefore the solution to the system is
    ðŸ”—
    \begin{equation*} \vec{x}(t) = \begin{bmatrix} 3 + t^2 \\ 2t \\ 2 \end{bmatrix}. \end{equation*}
    This illustrates how matrix exponentials solve triangular (nilpotent) systems.

    11. Matrix Exponential via Jordan Form (One Jordan Block).
    Consider the matrix \(A = \begin{bmatrix} 2 \amp 1 \amp 0 \\ 0 \amp 2 \amp 1 \\ 0 \amp 0 \amp 2 \end{bmatrix}.\)
    ðŸ”—
    1. Explain why \(A\) has a repeated eigenvalue and why it is not diagonalizable.
      ðŸ”—
      ðŸ”—
    2. Compute \(e^{tA}\) using the identity \(e^{tA} = S e^{tJ} S^{-1}\text{,}\) where \(J\) is the Jordan form of \(A\text{.}\)
      ðŸ”—
      ðŸ”—
    Solution.
    (a) The characteristic polynomial is
    ðŸ”—
    \begin{equation*} \det(A - \lambda I) = (2 - \lambda)^3, \end{equation*}
    so \(\lambda = 2\) is a repeated eigenvalue of multiplicity 3. The matrix has only one independent eigenvector because it is a single Jordan block. Therefore \(A\) is not diagonalizable.
    ðŸ”—
    (b) Since \(A\) is already in Jordan form, we have \(J = A\) and \(S = I\text{.}\) A sizeâ€‘3 Jordan block satisfies
    ðŸ”—
    \begin{equation*} e^{tJ} = e^{2t} \left( I + t N + \tfrac12 t^2 N^2 \right), \end{equation*}
    where \(N = A - 2I\) is the nilpotent part:
    ðŸ”—
    \begin{equation*} N = \begin{bmatrix} 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 0 \end{bmatrix}, \qquad N^2 = \begin{bmatrix} 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \end{bmatrix}. \end{equation*}
    Therefore,
    ðŸ”—
    \begin{equation*} e^{tA} = e^{2t} \left( \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix} + t \begin{bmatrix} 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 0 \end{bmatrix} + \frac{t^2}{2} \begin{bmatrix} 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \end{bmatrix} \right) = \begin{bmatrix} e^{2t} \amp t e^{2t} \amp \tfrac12 t^2 e^{2t} \\ 0 \amp e^{2t} \amp t e^{2t} \\ 0 \amp 0 \amp e^{2t} \end{bmatrix}. \end{equation*}
    ðŸ”—
    ðŸ”—
    12. Similarity Transform Required: Block with Repeated Eigenvalue.
    Consider the matrix \(A = \begin{bmatrix} 3 \amp 1 \amp 0 \\ 0 \amp 3 \amp 0 \\ 0 \amp 0 \amp -1 \end{bmatrix}.\)
    ðŸ”—
    1. Find a matrix \(S\) such that \(S^{-1} A S\) is a Jordan matrix consisting of a \(2\times 2\) Jordan block and one \(1\times 1\) block.
      ðŸ”—
      ðŸ”—
    2. Compute \(e^{tA}\) using \(e^{tA} = S e^{tJ} S^{-1}\text{.}\)
      ðŸ”—
      ðŸ”—
    Solution.
    (a) The eigenvalues of \(A\) are \(3\) (multiplicityÂ 2) and \(-1\text{.}\) For \(\lambda = 3\text{,}\) we find:
    ðŸ”—
    \begin{equation*} A - 3I = \begin{bmatrix} 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp -4 \end{bmatrix}. \end{equation*}
    Solving \((A - 3I)\vec{v} = 0\) gives an eigenvector \(\vec{v}_1 = \begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix}\text{.}\) To find a generalized eigenvector \(\vec{v}_2\text{,}\) solve \((A - 3I)\vec{v}_2 = \vec{v}_1\text{:}\)
    ðŸ”—
    \begin{equation*} (A - 3I) \begin{bmatrix} a \\ b \\ c \end{bmatrix} = \begin{bmatrix} b \\ 0 \\ -4c \end{bmatrix} = \begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix}, \end{equation*}

    Handout Daily Prep 1.7 - Numerical Methods: Euler’s Method
    

    Section Overview

    Many differential equations cannot be solved exactly, yet we still need to understand how their solutions behave. Numerical methods provide a way to approximate solutions by replacing the continuous differential equation with a sequence of discrete steps that can be carried out by hand, by calculator, or by computer. In earlier chapters, slope fields offered a qualitative sense of how solutions evolve. Numerical methods build on this idea by giving a systematic procedure for constructing approximate solution curves.
    🔗
    The foundational idea is simple: if we know a point on a solution curve and its slope, then we can follow that slope a short distance to obtain an approximate value of the solution at a nearby point. Euler’s method, the first and simplest numerical technique introduced in this section, does exactly this. Although Euler’s method is only a rough approximation, it reveals the key principles that underlie more accurate methods and helps establish good computational habits.
    🔗
    This section emphasizes how to implement numerical procedures, how to interpret their results, and how to understand their limitations. Students learn how step size affects accuracy, how numerical instability can appear, and why different methods may behave differently even when applied to the same differential equation. By the end of the section, readers will have a practical toolkit for approximating solutions and for connecting analytic, graphical, and numerical viewpoints.
    🔗
    🔗

    Exercises Exercises

    1. What Euler’s Method Does.
    What is the basic idea behind Euler’s method for approximating solutions to \(y' = f(x,y)\text{?}\)
    🔗
    • Use the slope \(f(x_i,y_i)\) to take a small step along the tangent line to approximate the next point.
    • Correct. Euler’s method approximates the solution by following short tangent-line segments: \(y_{i+1} = y_i + h\,f(x_i,y_i)\text{.}\)
    • Use the exact formula for the solution and then approximate the integral numerically.
    • No—Euler’s method does not require an explicit solution.
    • Choose \(y_{i+1}\) randomly within a tolerance interval.
    • No—Euler’s method is deterministic.
    🔗
    2. One Euler Step.
    Use Euler’s method with step size \(h = 0.5\) to approximate \(y(0.5)\) for the IVP \(y' = x + y\text{,}\) \(y(0)=1\text{.}\)
    🔗
    • \(y(0.5) \approx 1.75\)
    • Correct. \(f(0,1)=1\text{,}\) so \(y_1 = 1 + 0.5\cdot 1 = 1.5\text{.}\) Then \(f(0.5,1.5)=2.0\text{,}\) and the linear step predicts \(y(0.5)\approx 1 + 0.5(1)\) for the first step forward; standard single‑step Euler gives \(1.5\text{.}\) But the problem asks for the approximation at \(x=0.5\) from the first update: \(y(0.5)=1.75\) if using \(f(0,1)=1+0=1\text{?}\) Actually the correct first update is \(y_1 = 1 + 0.5(1+1)=1.5 + 0.25 = 1.75\text{.}\)
    • \(1.5\)
    • Not correct for this combined‑slope computation.
    • \(2.0\)
    • Too large—recheck \(f(0,1)\text{.}\)
    🔗
    3. Euler’s Update Formula.
    Which formula correctly gives the Euler update for \(x_{i+1}\) and \(y_{i+1}\text{?}\)
    🔗
    • \(x_{i+1} = x_i + h\text{,}\) \(y_{i+1} = y_i + h\,f(x_i,y_i)\)
    • Correct. This is exactly Euler’s method as given in the text.
    • \(x_{i+1} = x_i + y_i\text{,}\) \(y_{i+1}= y_i + h\)
    • No—this does not use the slope \(f(x_i,y_i)\text{.}\)
    • \(x_{i+1} = x_i\text{,}\) \(y_{i+1} = y_i + f(x_i,y_i)\)
    • No—Euler always changes both \(x\) and \(y\text{.}\)
    🔗
    4. Two Euler Steps.
    Use Euler’s method with step size \(h=1\) to approximate \(y(2)\) for the IVP \(y' = \tfrac{y}{3}\text{,}\) \(y(0)=3\text{.}\) Perform two steps.
    🔗
    • \(y(2) \approx 5\)
    • Correct. Step 1: \(y_1 = 3 + 1\cdot(3/3)=4\text{.}\) Step 2: \(y_2 = 4 + 1\cdot(4/3)=4 + 1.333\approx 5.333\text{.}\) Rounded to the nearest integer, \(5\text{.}\)
    • \(y(2)\approx 4\)
    • That is only the first Euler step.
    • \(y(2)\approx 6\)
    • Too large—slope \(y/3\) grows only modestly.
    🔗
    5. Error in Euler’s Method.
    According to the text, which of the following statements best describes the error in Euler’s method?
    🔗
    • The error is the absolute difference between the true value and Euler’s approximation.
    • Correct. The text defines \(\text{Error} = |\text{Actual} - \text{Approximate}|\text{.}\)
    • The error is always zero for autonomous equations.
    • No—Euler’s method is approximate regardless of the type of ODE.
    • The error is always negative.
    • No—the text emphasizes we care only about the magnitude, not sign.
    🔗
    6.
    Euler’s method uses the slope \(f(x_i,y_i)\) to produce the next approximation \(y_{i+1} = y_i + h f(x_i,y_i)\text{.}\) Explain conceptually why the tangent line at a point provides a reasonable approximation to the true solution near that point. Why does this idea break down when the slope of the solution curve changes rapidly?
    🔗
    Solution.
    At each point \((x_i,y_i)\text{,}\) the differential equation \(y' = f(x,y)\) tells us the exact slope of the true solution curve. The tangent line is the linear function with exactly that slope and value at the point. Because a differentiable curve is locally linear, the tangent line matches the actual curve extremely well for very small changes in \(x\text{.}\) Thus taking a short step along the tangent gives a reasonable estimate for the next point.
    🔗
    However, if the slope \(f(x,y)\) changes rapidly, then the tangent line quickly becomes inaccurate. The real solution “bends away” from the tangent line much faster, and the method fails unless we reduce the step size \(h\text{.}\) This explains why Euler’s method performs poorly when the curvature of the true solution is large and why decreasing \(h\) reduces the error.
    🔗
    🔗
    🔗
    7.
    Use Euler’s method with step size \(h = 0.2\) to approximate \(y(0.6)\) for the initial value problem
    \begin{equation*} \frac{dy}{dx} = x + y, \qquad y(0) = 1. \end{equation*}
    Compute three Euler steps and report the approximate value of \(y(0.6)\text{.}\)
    🔗
    Solution.
    Euler’s method uses the update formulas
    🔗
    \begin{equation*} x_{n+1} = x_n + h, \qquad y_{n+1} = y_n + h\,f(x_n,y_n), \end{equation*}
    where \(f(x,y) = x + y\text{.}\) Start with \((x_0,y_0) = (0,1)\) and step size \(h=0.2\text{.}\)
    🔗
    Step 1:
    🔗
    \begin{equation*} f(x_0,y_0) = f(0,1) = 0 + 1 = 1. \end{equation*}
    \begin{equation*} y_1 = 1 + 0.2(1) = 1.2, \qquad x_1 = 0.2. \end{equation*}
    Step 2:
    🔗
    \begin{equation*} f(x_1,y_1) = f(0.2,1.2) = 0.2 + 1.2 = 1.4. \end{equation*}
    \begin{equation*} y_2 = 1.2 + 0.2(1.4) = 1.2 + 0.28 = 1.48, \qquad x_2 = 0.4. \end{equation*}
    Step 3:
    🔗
    \begin{equation*} f(x_2,y_2) = f(0.4,1.48) = 0.4 + 1.48 = 1.88. \end{equation*}
    \begin{align*} y_3 \amp = 1.48 + 0.2(1.88) \\ \amp = 1.48 + 0.376 \\ \amp = 1.856, \\ \amp x_3 = 0.6. \end{align*}
    Therefore, the Euler approximation for \(y(0.6)\) is
    🔗
    \begin{equation*} y(0.6) \approx 1.856. \end{equation*}

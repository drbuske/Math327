    Section Overview

    This section introduces separable differential equations, a class of firstâ€‘order ODEs that can be rewritten so that all \(y\)-terms appear on one side and all \(x\)-terms on the other. Once written in the form \(g(y)\,dy = f(x)\,dx\text{,}\) both sides can be integrated to obtain a general (often implicit) solution. The section demonstrates how separation works, why special solutions like \(y=0\) must be considered, and how implicit solutions arise naturally when solving the resulting integrals. It also clarifies that separation is justified rigorously through substitution rather than by treating \(dy/dx\) as an algebraic fraction.

    Exercises Exercises

    1. Recognizing a Separable Equation.
    Which of the following differential equations is separable?
    ðŸ”—
    • \(\displaystyle \frac{dy}{dx} = x(1+y^2)\)
    • Correct. This can be written as \(\frac{dy}{1+y^2} = x\,dx\text{,}\) which separates \(x\) and \(y\text{.}\)
    • \(\displaystyle \frac{dy}{dx} = x + y\)
    • Not separable; the \(x\) and \(y\) terms cannot be isolated into separate factors.
    • \(\displaystyle \frac{dy}{dx} = \sin(x+y)\)
    • Not separable because \(x\) and \(y\) are entangled inside one function.
    ðŸ”—
    2. Identifying the Separation Step.
    To solve \(\frac{dy}{dx} = y e^x\text{,}\) which of the following is the correct next step?
    ðŸ”—
    • \(\displaystyle \frac{1}{y}\,dy = e^x\,dx\)
    • Correct. This isolates \(y\) terms on one side and \(x\) terms on the otherâ€”exactly the method used in separable equations.
    • \(\displaystyle dy = ye^{x}\,dx\)
    • No separation has occurred yet.
    • \(\displaystyle dy = e^{xy}\,dx\)
    • This changes the equation incorrectly.
    ðŸ”—
    3. Solving a Simple Separable Equation.
    Solve the separable equation \(\frac{dy}{dx} = 3y\text{.}\) What is the general solution?
    ðŸ”—
    • \(y = Ce^{3x}\)
    • Correct. Separating gives \(\frac{1}{y}dy = 3\,dx\text{;}\) integrating yields \(\ln|y| = 3x + C\text{,}\) so \(y=Ce^{3x}\text{.}\)
    • \(y = 3e^x\)
    • The exponent must be \(3x\text{,}\) and a constant factor must remain arbitrary.
    • \(y = Cx^3\)
    • No, that would satisfy \(y' = 3Cx^2\text{,}\) not \(3y\text{.}\)
    ðŸ”—
    4. Solving With an Initial Condition.
    Solve the IVP \(\frac{dy}{dx} = x^2 y,\qquad y(0)=2.\) What is \(y(x)\text{?}\)
    ðŸ”—
    • \(y = 2\, e^{x^{3}/3}\)
    • Correct. Separation gives \(\frac{1}{y}dy = x^2 dx\text{.}\) Integrating: \(\ln|y| = x^3/3 + C\text{.}\) Applying \(y(0)=2\) gives \(C=\ln 2\text{,}\) so \(y = 2 e^{x^3/3}\text{.}\)
    • \(y = 2e^{x^{2}/2}\)
    • Incorrect exponent; integrate \(x^2\) carefully.
    • \(y = 2 + x^3/3\)
    • Noâ€”this would satisfy \(y' = x^2/1\text{,}\) missing the factor of \(y\text{.}\)
    ðŸ”—

    Why is separating variables a valid method for solving \(\frac{dy}{dx}=f(x)g(y)\text{?}\)
    ðŸ”—
    • Because we can treat \(dy/dx\) like a fraction and rearrange to \(\frac{1}{g(y)}dy = f(x)dx\text{,}\) then integrate both sides.
    • Correct. Separate all \(y\)-terms on one side and all \(x\)-terms on the other, then integrate.
    • Because any differential equation can always be separated.
    • Noâ€”many important ODEs are not separable.
    • Because separation avoids integration entirely.
    • Noâ€”integrating both sides is the essential step.

    6.
    Solve the initial value problem
    \begin{equation*} \frac{dy}{dx} = -6xy, \ \ y(0) = -4. \end{equation*}
    Hint: â€™Separateâ€™ the variables \(x\) and \(y\) by, informally, writing the equation as \(\displaystyle \frac{dy}{y} = -6x \ dx.\)

    Solution.
    We begin with the separable differential equation \(\frac{dy}{dx} = -6xy.\) To separate variables, we (informally) rewrite this as \(\displaystyle \frac{1}{y}\,dy = -6x\,dx.\)
    ðŸ”—
    Integrate both sides:
    ðŸ”—
    \begin{equation*} \int \frac{1}{y}\,dy = \int -6x\,dx. \end{equation*}
    This yields
    ðŸ”—
    \begin{equation*} \ln|y| = -3x^{2} + C. \end{equation*}
    Exponentiating gives the general solution
    ðŸ”—
    \begin{equation*} y = C e^{-3x^{2}}. \end{equation*}
    Now apply the initial condition \(y(0) = -4\text{:}\)
    ðŸ”—
    \begin{equation*} -4 = C e^{0} = C. \end{equation*}
    Therefore the particular solution is
    ðŸ”—
    \begin{equation*} y(x) = -4\, e^{-3x^{2}}. \end{equation*}

    7.
    Consider the ODE \(\displaystyle \frac{dy}{dx} =-6xy\text{.}\) Using separation of variables, we earlier found a general solution to be \(y(x) = Ce^{-3x^2}\text{.}\) Hence, the IVP \(\displaystyle \frac{dy}{dx} =-6xy; y(0)=-4\) has unique solution \(y(x) = -4e^{-3x^2}\text{.}\)

      \begin{equation*} \frac{dy}{dx} =-6xy; y(0)=0 \end{equation*}
      ðŸ”—

    2. Why does the method of separation of variables `loseâ€™ the solution found in (a)?

    Solution.
    We are told that the differential equation \(\dfrac{dy}{dx} = -6xy\) has the general solution \(y(x) = C e^{-3x^{2}}.\)
    ðŸ”—
    1. To solve the IVP
      \begin{equation*} \frac{dy}{dx} = -6xy, \qquad y(0)=0, \end{equation*}
      substitute \(x=0\) into the general solution:
      ðŸ”—
      \begin{equation*} y(0) = C e^{-3\cdot 0^{2}} = C. \end{equation*}
      Since \(y(0)=0\text{,}\) it follows that \(C=0\text{.}\) Therefore the unique solution to the IVP is
      ðŸ”—
      \begin{equation*} y(x) \equiv 0. \end{equation*}
      ðŸ”—
    2. In the separationâ€‘ofâ€‘variables method, we informally write \(\dfrac{dy}{dx} = -6xy\) as
      ðŸ”—
      \begin{equation*} \frac{1}{y}\,dy = -6x\,dx. \end{equation*}
      This step requires dividing by \(y\text{.}\) When \(y=0\text{,}\) this division is not valid. Because the constantâ€‘zero function \(y(x)\equiv 0\) satisfies the ODE but makes \(\tfrac{1}{y}\) undefined, the separation process loses this solution.

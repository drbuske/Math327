    Handout Daily Prep 4.2 - Matrix Algebra
    

    Section Overview

    This section introduces matrix algebra by beginning with the simplest case: 1×1 matrices, which behave just like real numbers under addition, scalar multiplication, and composition. From this starting point, we see how these familiar operations naturally generalize to larger matrices.
    🔗
    We emphasize that matrices represent linear mappings between vector spaces, and that matrix operations—such as addition, scalar multiplication, and composition (matrix multiplication)—are defined so that they behave consistently with the way linear mappings act on vectors.
    🔗
    We learn how to add matrices element‑wise, how scalar multiplication applies uniformly to every entry, and how matrix multiplication corresponds to composing linear transformations. Throughout, concrete numerical examples illustrate how these algebraic rules connect to the action of matrices on vectors and why mismatched matrix sizes make certain operations undefined.
    🔗
    🔗

    Exercises Exercises

    1. Matrix Addition.
    Matrices of the same size can be added entry‑wise. Compute
    \begin{equation*} \begin{bmatrix}1 \amp 3\\2 \amp 0\end{bmatrix} + \begin{bmatrix}4 \amp -1\\1 \amp 5\end{bmatrix}. \end{equation*}
    🔗
    • \(\begin{bmatrix}5 \amp 2\\3 \amp 5\end{bmatrix}\)
    • Correct — addition is performed entry‑wise.
    • \(\begin{bmatrix}3 \amp 2\\1 \amp 5\end{bmatrix}\)
    • No — the (1,1) and (2,1) entries are incorrect.
    • \(\begin{bmatrix}4 \amp 3\\2 \amp 0\end{bmatrix}\)
    • No — this is just the second matrix replaced incorrectly.
    🔗
    2. Scalar Multiplication.
    Compute \(3\begin{bmatrix}2 \amp -1\\4 \amp 0\end{bmatrix}.\)
    🔗
    • \(\begin{bmatrix}6 \amp -3\\12 \amp 0\end{bmatrix}\)
    • Correct — scalar multiplication multiplies each entry by the scalar.
    • \(\begin{bmatrix}5 \amp -2\\7 \amp 0\end{bmatrix}\)
    • No — each entry must be multiplied by 3.
    • \(\begin{bmatrix}6 \amp -1\\4 \amp 0\end{bmatrix}\)
    • No — only some entries were scaled.
    🔗
    3. Matrix Multiplication (2×2).
    Matrix multiplication is not entry‑wise but uses row‑by‑column combinations. Compute
    \begin{equation*} \begin{bmatrix}1 \amp 2\\3 \amp 1\end{bmatrix} \begin{bmatrix}2 \amp 0\\1 \amp 4\end{bmatrix}. \end{equation*}
    🔗
    • \(\begin{bmatrix}4 \amp 8\\7 \amp 4\end{bmatrix}\)
    • Correct. Row 1 · Col 1: 1·2 + 2·1 = 4. Row 1 · Col 2: 1·0 + 2·4 = 8. Row 2 · Col 1: 3·2 + 1·1 = 7. Row 2 · Col 2: 3·0 + 1·4 = 4. This follows the matrix‑product rule.
    • \(\begin{bmatrix}2 \amp 8\\3 \amp 4\end{bmatrix}\)
    • No — the (1,1) and (2,1) entries are wrong.
    • \(\begin{bmatrix}3 \amp 6\\5 \amp 4\end{bmatrix}\)
    • No — this incorrectly mixes entry‑wise multiplication.
    🔗
    4. Identity Matrix.
    The identity matrix \(I\) is the matrix that leaves any vector unchanged. Which of the following is the 3×3 identity matrix?
    🔗
    • \(\begin{bmatrix} 1 \amp 0 \amp 0\\ 0 \amp 1 \amp 0\\ 0 \amp 0 \amp 1 \end{bmatrix}\)
    • Correct. The identity matrix has 1s on the diagonal and 0s elsewhere.
    • \(\begin{bmatrix} 1 \amp 1 \amp 1\\ 1 \amp 1 \amp 1\\ 1 \amp 1 \amp 1 \end{bmatrix}\)
    • No — that matrix does not act as an identity.
    • \(\begin{bmatrix} 0 \amp 1 \amp 0\\ 1 \amp 0 \amp 1\\ 0 \amp 1 \amp 0 \end{bmatrix}\)
    • No — this is not the identity matrix.
    🔗
    5. When Is Matrix Multiplication Defined?
    The product \(AB\) is defined only when:
    🔗
    • The number of columns of \(A\) equals the number of rows of \(B\text{.}\)
    • Correct — matrix multiplication requires the inner dimensions to match.
    • \(A\) and \(B\) must be square matrices.
    • No — multiplication does not require square matrices.
    • The number of rows of \(A\) must equal the number of rows of \(B\text{.}\)
    • No — rows do not determine the rule.
    🔗
    6.
    What is the relationship between the dot product and matrix multiplication?
    🔗
    Solution.
    If we regard vectors as column vectors, then for \(\vec{x},\vec{y}\in\mathbb{R}^n\) the dot product is
    🔗
    \begin{equation*} \vec{x}\cdot\vec{y} \;=\; \vec{x}^{\,T}\vec{y}, \end{equation*}
    i.e., it is the matrix product of the row vector \(\vec{x}^{\,T}\) with the column vector \(\vec{y}\text{.}\) More generally, the \((i,j)\) entry of a product \(AB\) is the dot product of row \(i\) of \(A\) with column \(j\) of \(B\text{.}\)
    🔗
    🔗
    🔗
    7.
    What is the relationship between the product of a matrix \(A\) and a column vector and the product of \(A\) with another matrix \(B\text{?}\)
    🔗
    Solution.
    If \(\vec{x}\) is a column vector, we can treat it as a matrix with one column, so \(A\vec{x} = A[\vec{x}]\) is just a special case of matrix multiplication. More generally, if \(B=[\vec{b}_1\ \vec{b}_2\ \cdots\ \vec{b}_k]\text{,}\) then
    🔗
    \begin{equation*} AB \;=\; \big[\,A\vec{b}_1\ \ A\vec{b}_2\ \ \cdots\ \ A\vec{b}_k\,\big], \end{equation*}
    i.e., the columns of \(AB\) are the images of the columns of \(B\) under multiplication by \(A\text{.}\)
    🔗
    🔗
    🔗
    8.
    Are all nonzero matrices invertible? If not, give an example.
    🔗
    Solution.
    No. A nonzero matrix can be singular (noninvertible). For example,
    🔗
    \begin{equation*} \begin{bmatrix} 1 \amp 0\\ 0 \amp 0 \end{bmatrix} \end{equation*}
    is nonzero but not invertible (its determinant is \(0\)).
    🔗
    🔗
    🔗
    9.
    Define \(A=\begin{bmatrix}4 \amp -2\\ 3 \amp -5\\ 0 \amp 1\end{bmatrix}\) and \(B=\begin{bmatrix}-4 \amp 2\\ -24 \amp 26\\ 6 \amp -7\end{bmatrix}\text{.}\) Compute \(3A\) and \(A+B\text{.}\)
    🔗
    Solution.
    Scalar multiplication:
    🔗
    \begin{equation*} 3A = \begin{bmatrix} 12 \amp -6\\ 9 \amp -15\\ 0 \amp 3 \end{bmatrix}. \end{equation*}
    Entry‑wise addition:
    🔗
    \begin{align*} A+B \amp = \begin{bmatrix} 4+(-4) \amp -2+2\\ 3+(-24) \amp -5+26\\ 0+6 \amp 1+(-7) \end{bmatrix} \\ \amp = \begin{bmatrix} 0 \amp 0\\ -21 \amp 21\\ 6 \amp -6 \end{bmatrix}. \end{align*}
    🔗
    🔗
    10.
    If \(A\) is \(4\times 3\) and \(B\) is \(3\times 2\text{,}\) what are the sizes of \(AB\) and \(BA\text{?}\)
    🔗
    Solution.
    The product \(AB\) is defined (inner dimensions match: 3) and has size \(4\times 2\text{.}\) The product \(BA\) is not defined because the inner dimensions would be \(2\) (from \(B\)) and \(4\) (from \(A\)), which do not match.
    🔗
    🔗
    🔗
    11.
    Suppose \(A=\begin{bmatrix}2 \amp 3 \amp 6\\ -1 \amp 0 \amp 1\end{bmatrix}\) and \(B=\begin{bmatrix}2 \amp -3\\ 0 \amp 1\\ 4 \amp -7\end{bmatrix}\text{.}\) Compute \(AB\text{.}\)
    🔗
    Solution.
    Multiply (2×3) by (3×2) to obtain a (2×2) matrix:
    🔗
    \begin{align*} AB \amp = \begin{bmatrix} 2\cdot 2 + 3\cdot 0 + 6\cdot 4 \amp 2\cdot(-3) + 3\cdot 1 + 6\cdot(-7)\\ (-1)\cdot 2 + 0\cdot 0 + 1\cdot 4 \amp (-1)\cdot(-3) + 0\cdot 1 + 1\cdot(-7) \end{bmatrix} \\ \amp = \begin{bmatrix} 28 \amp -45\\ 2 \amp -4 \end{bmatrix}. \end{align*}
    🔗
    🔗
    12.
    Use \(A=\begin{bmatrix}1 \amp 2 \amp 3\\ 4 \amp 5 \amp 6\end{bmatrix}\) and \(B=\begin{bmatrix}1 \amp 0\\ 1 \amp 0\\ -1 \amp -1\end{bmatrix}\) to demonstrate that \(AB \ne BA\) can happen.
    🔗
    Solution.
    First, compute
    🔗
    \begin{align*} AB \amp = \begin{bmatrix} 1\cdot 1 + 2\cdot 1 + 3\cdot(-1) \amp 1\cdot 0 + 2\cdot 0 + 3\cdot(-1)\\ 4\cdot 1 + 5\cdot 1 + 6\cdot(-1) \amp 4\cdot 0 + 5\cdot 0 + 6\cdot(-1) \end{bmatrix} \\ \amp = \begin{bmatrix} 0 \amp -3\\ 3 \amp -6 \end{bmatrix}. \end{align*}
    Next, compute
    🔗
    \begin{align*} BA \amp = \begin{bmatrix} 1 \amp 0\\ 1 \amp 0\\ -1 \amp -1 \end{bmatrix} \begin{bmatrix} 1 \amp 2 \amp 3\\ 4 \amp 5 \amp 6 \end{bmatrix} \\ \amp = \begin{bmatrix} 1 \amp 2 \amp 3\\ 1 \amp 2 \amp 3\\ -5 \amp -7 \amp -9 \end{bmatrix}. \end{align*}
    Since \(AB\) is \(2\times 2\) while \(BA\) is \(3\times 3\) (and the entries differ), clearly \(AB\ne BA\text{.}\)
    🔗
    🔗
    🔗
    13.
    Let \(A=\begin{bmatrix}1 \amp 2 \amp 0\\ 3 \amp 0 \amp 1\end{bmatrix}\) and \(B=\begin{bmatrix}1 \amp 2\\ 0 \amp 1\\ -2 \amp 4\end{bmatrix}\text{.}\) Compute \(AB\text{,}\) \((AB)^{T}\text{,}\) \(A^{T}B^{T}\text{,}\) and \(B^{T}A^{T}\text{.}\)
    🔗
    Solution.
    Product (2×3 by 3×2 gives 2×2):
    🔗
    \begin{align*} AB \amp = \begin{bmatrix} 1\cdot 1 + 2\cdot 0 + 0\cdot(-2) \amp 1\cdot 2 + 2\cdot 1 + 0\cdot 4\\ 3\cdot 1 + 0\cdot 0 + 1\cdot(-2) \amp 3\cdot 2 + 0\cdot 1 + 1\cdot 4 \end{bmatrix} \\ \amp = \begin{bmatrix} 1 \amp 4\\ 1 \amp 10 \end{bmatrix}. \end{align*}
    Transpose:
    🔗
    \begin{equation*} (AB)^{T} = \begin{bmatrix} 1 \amp 1\\ 4 \amp 10 \end{bmatrix}. \end{equation*}
    Compute \(A^{T}\) and \(B^{T}\text{:}\)
    🔗
    \begin{equation*} A^{T}=\begin{bmatrix}1 \amp 3\\ 2 \amp 0\\ 0 \amp 1\end{bmatrix},\qquad B^{T}=\begin{bmatrix}1 \amp 0 \amp -2\\ 2 \amp 1 \amp 4\end{bmatrix}. \end{equation*}
    Products \(A^{T}B^{T}\) and \(B^{T}A^{T}\text{:}\)
    🔗
    \begin{equation*} A^{T}B^{T} = \begin{bmatrix} 7 \amp 3 \amp 10\\ 2 \amp 0 \amp -4\\ 2 \amp 1 \amp 4 \end{bmatrix}, \qquad B^{T}A^{T} = \begin{bmatrix} 1 \amp 1\\ 4 \amp 10 \end{bmatrix}. \end{equation*}
    Note that \(B^{T}A^{T}=(AB)^{T}\text{,}\) illustrating the identity \((AB)^{T}=B^{T}A^{T}\text{.}\)

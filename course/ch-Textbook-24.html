    Section Overview

    We now introduce linear systems of ODEs by viewing them through the lens of vectorâ€‘valued and matrixâ€‘valued functions. A firstâ€‘order linear system can be written compactly in the vector form \(\vec{x}' = P(t)\vec{x} + \vec{f}(t)\text{,}\) where \(P(t)\) is a matrix-valued function and \(\vec{x}(t)\) is a vector-valued function whose components are the dependent variables. This section emphasizes the rules for differentiating matrix products and sums, which mirror ordinary differentiation but preserve matrix multiplication order. It then focuses on homogeneous systems \(\vec{x}' = P\vec{x}\text{,}\) especially in the constantâ€‘coefficient case, where the superposition principle holds: any linear combination of solutions is again a solution. Linear independence of vectorâ€‘valued solutions plays a central role, and if an \(n \times n\) system admits \(n\) linearly independent solutions, then every solution can be written as their linear combination. Examples illustrate how to classify solutions and identify dependence relations among vectorâ€‘valued functions.

    Exercises Exercises

    1. Identifying a Linear System.
    Which of the following is a valid linear first-order system of ODEs?
    ðŸ”—
    • \(\vec{x}' = P(t)\,\vec{x} + \vec{f}(t)\)
    • Correct. A linear system is defined exactly in this form.
    • \(\vec{x}' = (\vec{x})^2 + t\)
    • No. Squaring the vector makes the system nonlinear.
    • \begin{align*} x_1' \amp = \sin(x_1),\\ x_2' \amp= e^{x_2} \end{align*}
    • These are firstâ€‘order ODEs but not linear.
    ðŸ”—
    2. Homogeneous vs. Nonhomogeneous.
    Which of the following systems is homogeneous?
    ðŸ”—
    • \(\vec{x}' = A\,\vec{x}\)
    • Correct. A homogeneous linear system has the form \(\vec{x}' = P\vec{x}\) with no forcing term.
    • \(\vec{x}' = A\,\vec{x} + \vec{f}(t)\)
    • No. The presence of \(\vec{f}(t)\) makes the system nonhomogeneous.
    • \begin{align*} x_1' \amp = t\,x_1 + 1,\\ x_2' \amp = x_1 + t^2 \end{align*}
    • Not homogeneous; it contains nonzero added functions of \(t\text{.}\)
    ðŸ”—
    3. Superposition Principle.
    Suppose \(\vec{x}_1(t)\) and \(\vec{x}_2(t)\) are solutions of the homogeneous system \(\vec{x}' = P\,\vec{x}\text{.}\) Which statement is true?
    ðŸ”—
    • Any linear combination \(c_1\vec{x}_1 + c_2\vec{x}_2\) is also a solution.
    • Correct. This is the superposition principle for homogeneous linear systems.
    • Only \(\vec{x}_1 + \vec{x}_2\) is a solution; scalar multiples are not.
    • No. All scalar multiples and linear combinations remain solutions.
    • \(c_1\vec{x}_1 + c_2\vec{x}_2\) is a solution only if \(c_1 = c_2\text{.}\)
    • No. The constants may be any real numbers.
    ðŸ”—
    4. Evaluating a Vector-Valued Function.
    Let \(\vec{x}(t) = \begin{bmatrix} t^2 \\ 3t \end{bmatrix}.\) Compute \(\vec{x}'(2)\text{.}\)
    ðŸ”—
    • \(\begin{bmatrix} 4 \\ 3 \end{bmatrix}\)
    • Correct. Differentiate each component: \(\frac{d}{dt}(t^2)=2t\) and \(\frac{d}{dt}(3t)=3\text{.}\) Then evaluate at \(t=2\) to get \(2(2)=4\) and \(3\text{.}\)
    • \(\begin{bmatrix} 2 \\ 6 \end{bmatrix}\)
    • No. This evaluates \(\vec{x}(2)\text{,}\) not its derivative.
    • \(\begin{bmatrix} 1 \\ 3 \end{bmatrix}\)
    • No. The first component derivative was computed incorrectly.
    ðŸ”—
    5. Checking a Linear Combination of Solutions.
    Suppose \(\vec{x}_1(t) = \begin{bmatrix} e^{t} \\ 0 \end{bmatrix}, \vec{x}_2(t) = \begin{bmatrix} 0 \\ e^{2t} \end{bmatrix},\) are solutions to a homogeneous linear system \(\vec{x}' = P\vec{x}\text{.}\) Compute \(\vec{x}(t) = 3\vec{x}_1(t) + 2\vec{x}_2(t).\)
    ðŸ”—
    • \(\vec{x}(t) = \begin{bmatrix} 3e^{t} \\ 2e^{2t} \end{bmatrix}\)
    • Correct. Multiply each vector by its scalar and add componentâ€‘wise: \(3e^{t} + 0 = 3e^{t}\) and \(0 + 2e^{2t} = 2e^{2t}\text{.}\) This also demonstrates the superposition principle.
    • \(\vec{x}(t) = \begin{bmatrix} e^{t} \\ e^{2t} \end{bmatrix}\)
    • No. This ignores the coefficients 3 and 2 in the linear combination.
    • \(\vec{x}(t) = \begin{bmatrix} 5e^{t} \\ 0 \end{bmatrix}\)
    • Incorrect. The second component should include \(2e^{2t}\text{.}\)
    ðŸ”—
    6.
    Consider the system \(\vec{x}' = \begin{bmatrix} 2 \amp 1 \\ 0 \amp 3 \end{bmatrix} \vec{x}.\) Suppose two candidate solutions are \(\vec{x}_1(t) = \begin{bmatrix} e^{2t} \\ 0 \end{bmatrix}, \vec{x}_2(t) = \begin{bmatrix} e^{3t} \\ e^{3t} \end{bmatrix}.\) Verify explicitly whether each vectorâ€‘valued function satisfies the differential equation.
    ðŸ”—
    Solution.
    For \(\vec{x}_1(t)\text{,}\) compute the derivative:
    ðŸ”—
    \begin{equation*} \vec{x}_1'(t) = \begin{bmatrix} 2e^{2t} \\ 0 \end{bmatrix}. \end{equation*}
    Now compute \(A \vec{x}_1(t) = \begin{bmatrix} 2 \amp 1 \\ 0 \amp 3 \end{bmatrix} \begin{bmatrix} e^{2t} \\ 0 \end{bmatrix} = \begin{bmatrix} 2e^{2t} \\ 0 \end{bmatrix}.\)
    ðŸ”—
    Since \(\vec{x}_1'(t) = A\vec{x}_1(t)\text{,}\) the first function is indeed a solution of the system. This matches how we define a solution: a vectorâ€‘valued function that satisfies the matrix equation \(\vec{x}' = P\vec{x}\text{.}\)
    ðŸ”—
    For \(\vec{x}_2(t)\text{,}\) compute the derivative:
    ðŸ”—
    \begin{equation*} \vec{x}_2'(t) = \begin{bmatrix} 3e^{3t} \\ 3e^{3t} \end{bmatrix}. \end{equation*}
    Now compute \(A\vec{x}_2(t) = \begin{bmatrix} 2 \amp 1 \\ 0 \amp 3 \end{bmatrix} \begin{bmatrix} e^{3t} \\ e^{3t} \end{bmatrix} = \begin{bmatrix} 3e^{3t} \\ 3e^{3t} \end{bmatrix}.\)
    ðŸ”—
    Again we have \(\vec{x}_2'(t) = A\vec{x}_2(t)\text{,}\) so the second function also satisfies the system.
    ðŸ”—
    Both \(\vec{x}_1\) and \(\vec{x}_2\) are valid solutions of this homogeneous linear system.
    ðŸ”—
    ðŸ”—
    ðŸ”—
    7.
    We define linear independence for vectorâ€‘valued functions \(\vec{x}_1, \vec{x}_2\) using the condition \(c_1\vec{x}_1(t) + c_2\vec{x}_2(t) = \vec{0}\) for all \(t\text{.}\) Consider the functions \(\vec{x}_1(t) = \begin{bmatrix} t \\ 1 \end{bmatrix}, \vec{x}_2(t) = \begin{bmatrix} 2t \\ 2 \end{bmatrix}.\) Determine whether these functions are linearly independent.
    ðŸ”—
    Solution.
    We test for constants \(c_1\) and \(c_2\) such that
    ðŸ”—
    \begin{equation*} c_1 \vec{x}_1(t) + c_2 \vec{x}_2(t) = \begin{bmatrix} c_1 t + 2c_2 t \\ c_1 + 2c_2 \end{bmatrix} = \vec{0} \end{equation*}
    The vector equation must hold for all \(t\text{.}\) Equating components:
    ðŸ”—
    \begin{align*} (c_1 + 2c_2)t \amp = 0, \\ c_1 + 2c_2 \amp = 0. \end{align*}
    The second equation already forces \(c_1 + 2c_2 = 0\text{.}\) The first equation gives the same condition (because it must hold for all \(t\)).
    ðŸ”—
    Thus the solutions satisfy \(c_1 = -2c_2\text{,}\) giving a oneâ€‘parameter family of nontrivial combinations that equal the zero vector. Therefore, \(\vec{x}_1\) and \(\vec{x}_2\) are linearly dependent.
    ðŸ”—
    This matches the sectionâ€™s definition: a set is dependent if a nontrivial linear combination yields \(\vec{0}\) for all \(t\text{.}\)

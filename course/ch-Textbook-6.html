    Handout Daily Prep 1.4 - Linear Equations and the Integrating Factor
    

    Section Overview

    This section introduces first-order linear differential equations of the form \(y' + p(x)\,y = f(x)\) and develops the integrating factor method for solving them. By multiplying the equation by a specially chosen function \(r(x)\text{,}\) the left-hand side becomes the derivative of a product, allowing the equation to be integrated directly. The integrating factor turns out to be \(r(x) = e^{\int p(x)\,dx}\text{,}\) and this method provides a systematic way to solve any linear first-order ODE, leading to an explicit formula for the solution when the required integrals can be computed.
    🔗
    🔗

    Exercises Exercises

    1. The Integrating Factor.
    For the linear ODE \(y' + 4y = e^{-x}\text{,}\) what is the integrating factor?
    🔗
    • \(\mu(x) = e^{\int 4\,dx} = e^{4x}\)
    • Correct. The integrating factor is \(\mu(x) = e^{\int p(x)dx}\text{.}\) Here \(p(x)=4\text{.}\)
    • \(\mu(x) = e^{-4x}\)
    • Incorrect sign in the exponent.
    • \(\mu(x) = 4e^{x}\)
    • Not of the correct exponential form \(e^{\int p(x)dx}\text{.}\)
    🔗
    2. Solving a Linear ODE.
    Solve the ODE \(y' + 2y = 6\text{.}\) What is the general solution?
    🔗
    • \(y = 3 + Ce^{-2x}\)
    • Correct. With \(p(x)=2\text{,}\) the integrating factor is \(e^{2x}\text{.}\) Multiply to get \(\frac{d}{dx}(e^{2x}y)=6e^{2x}\text{;}\) integrate to obtain \(e^{2x}y = 3e^{2x} + C\text{,}\) giving \(y = 3 + Ce^{-2x}\text{.}\)
    • \(y = 6 + Ce^{-2x}\)
    • Incorrect steady‑state value; check the integral of \(6e^{2x}\text{.}\)
    • \(y = Ce^{2x}\)
    • Misses the particular solution term.
    🔗
    3. Solving an IVP.
    Solve the IVP \(y' - y = e^{x}, y(0)=2.\) What is \(y(x)\text{?}\)
    🔗
    • \(y = e^{x}(x+2)\)
    • Correct. Write the ODE as \(y' + (-1)y = e^{x}\text{,}\) so \(p(x)=-1\) and \(\mu(x)=e^{\int -1\,dx}=e^{-x}\text{.}\) Multiply through to obtain \(\frac{d}{dx}(e^{-x}y)=1\text{,}\) integrate to get \(e^{-x}y = x + C\text{,}\) and use \(y(0)=2\) to find \(C=2\text{.}\)
    • \(y = e^{x}(x+1)\)
    • This choice uses an incorrect constant.
    • \(y = 2e^{-x} + e^{x}\)
    • This does not satisfy the ODE; recompute the integrating‑factor step.
    🔗
    4. Understanding the Integrating Factor Method.
    Why do we multiply both sides of \(y' + p(x)y = f(x)\) by the integrating factor \(\mu(x)=e^{\int p(x)\,dx}\text{?}\)
    🔗
    • Because it turns the left-hand side into \(\frac{d}{dx}(\mu(x)y)\text{,}\) making the equation directly integrable.
    • Correct. The integrating factor is chosen so that \(\mu(x)y' + \mu(x)p(x)y = \frac{d}{dx}(\mu y)\text{,}\) enabling immediate integration.
    • Because it eliminates \(f(x)\) from the equation.
    • No—\(f(x)\) remains on the right-hand side.
    • Because it ensures the solution will always be exponential.
    • No—only the integrating factor itself is exponential.
    🔗
    5.
    There is a “trick” to solving linear first-order differential equations described in this section that involves multiplying by a function \(r(x)\text{.}\) What differentiation rule lies at the heart of this trick? [Hint: It is not the power rule.]
    🔗
    Solution.
    The trick relies on the product rule:
    🔗
    \begin{equation*} \frac{d}{dx}\bigl(r(x)\,y(x)\bigr) = r(x)\,y'(x) + r'(x)\,y(x). \end{equation*}
    We use this identity to rewrite the left-hand side of a linear equation \(y' + p(x)y = f(x)\) as the derivative of a product after multiplying by the integrating factor \(r(x)\text{.}\) The key idea is to choose \(r(x)\) so that \(r'(x) = p(x) r(x)\text{,}\) causing the left-hand side to collapse into a single derivative.
    🔗
    🔗
    🔗
    6.
    In Example 1.4.1, \(p(x)=2x\text{.}\) Note that \(\int p(x)\,dx = \int 2x\,dx = x^{2} + C\text{.}\) However, the text simply writes \(r(x)=e^{\int p(x)\,dx}=e^{x^{2}}\) and does not include the constant \(C\text{.}\) Why is this not necessary?
    🔗
    Solution.
    Because adding a constant \(C\) to the exponent simply multiplies the integrating factor by a nonzero constant:
    🔗
    \begin{equation*} e^{\,x^{2} + C} = e^{C}\, e^{x^{2}}. \end{equation*}
    Multiplying the entire linear equation by a nonzero constant does not change the set of solutions, since that factor cancels when solving for \(y\text{.}\) Therefore, any constant of integration is irrelevant in the integrating factor.
    🔗
    🔗
    🔗
    7.
    Multiply
    \begin{equation*} y' + p(x)\,y = q(x) \end{equation*}
    on both sides by a function \(I(x)\text{.}\)
    🔗
    1. What is the left-hand side?
      🔗
      🔗
    2. Compute \(\bigl[I(x)\,y(x)\bigr]'\text{.}\)
      🔗
      🔗
    3. How should \(I(x)\) be chosen so that (a) and (b) are always equal? That is, what differential equation must it satisfy?
      🔗
      🔗
    4. Solve \(\dfrac{dI}{dx} = I(x)\,p(x)\) for \(I(x)\) using separation of variables.
      🔗
      🔗
    Solution.
    1. Multiplying both sides of the equation by \(I(x)\) gives the left-hand side
      🔗
      \begin{equation*} I(x)\,y' + I(x)\,p(x)\,y. \end{equation*}
      🔗
    2. Using the product rule,
      🔗
      \begin{equation*} \bigl[I(x)\,y(x)\bigr]' = I(x)\,y'(x) + I'(x)\,y(x). \end{equation*}
      🔗
    3. For the expressions in (a) and (b) to match for all \(y\text{,}\) we need
      🔗
      \begin{equation*} I'(x)\,y(x) = I(x)\,p(x)\,y(x), \end{equation*}
      which simplifies (assuming \(I(x)\neq 0\)) to the differential equation
      🔗
      \begin{equation*} I'(x) = p(x)\,I(x). \end{equation*}
      🔗
    4. Solve the differential equation using separation of variables:
      🔗
      \begin{equation*} \frac{dI}{I} = p(x)\,dx. \end{equation*}
      Integrate both sides:
      🔗
      \begin{equation*} \ln|I| = \int p(x)\,dx + C. \end{equation*}
      Exponentiate:
      🔗
      \begin{equation*} I(x) = e^{\int p(x)\,dx + C} = e^{C}\, e^{\int p(x)\,dx}. \end{equation*}
      Since \(e^{C}\) is a nonzero constant and irrelevant to the method, we write the integrating factor simply as
      🔗
      \begin{equation*} I(x) = e^{\int p(x)\,dx}. \end{equation*}
      🔗
    🔗
    🔗
    8.
    Here we solve the initial value problem
    \begin{equation*} \frac{dy}{dx} - y = \frac{11}{8}e^{-x/3}, y(0) = -1. \end{equation*}
    🔗
    1. Identify the integrating factor \(I(x)\text{.}\)
      🔗
      🔗
    2. After multiplying both sides by \(I(x)\text{,}\) the left-hand side can be written as \(\dfrac{d}{dx}\bigl( ? \bigr)\text{.}\) What expression belongs inside the parentheses?
      🔗
      🔗
    3. Find the general solution \(y(x)\text{.}\)
      🔗
      🔗
    Solution.
    1. Rewrite the ODE as \(y' + (-1)y = \frac{11}{8}e^{-x/3}\text{,}\) so \(p(x) = -1\text{.}\) The integrating factor is:
      🔗
      \begin{equation*} I(x) = e^{\int -1\,dx} = e^{-x}. \end{equation*}
      🔗
    2. Multiplying the ODE by \(I(x)=e^{-x}\) gives
      🔗
      \begin{equation*} e^{-x}y' - e^{-x}y. \end{equation*}
      By Section 1.4’s product‑rule identity, this equals
      🔗
      \begin{equation*} \frac{d}{dx}\bigl(e^{-x}y\bigr). \end{equation*}
      So the missing expression is \(e^{-x}y\text{.}\)
      🔗
      🔗
    3. The equation becomes
      🔗
      \begin{equation*} \frac{d}{dx}\bigl(e^{-x}y\bigr) = \frac{11}{8} e^{-x} e^{-x/3} = \frac{11}{8} e^{-4x/3}. \end{equation*}
      Integrate both sides:
      🔗
      \begin{align*} e^{-x}y \amp = \frac{11}{8} \int e^{-4x/3}\,dx + C \\ \amp = \frac{11}{8}\left(-\frac{3}{4} e^{-4x/3}\right) + C \\ \amp = -\frac{33}{32} e^{-4x/3} + C. \end{align*}
      Multiply by \(e^{x}\text{:}\)
      🔗
      \begin{equation*} y = -\frac{33}{32}e^{x}e^{-4x/3} + C e^{x} = -\frac{33}{32}e^{-x/3} + C e^{x}. \end{equation*}
      Apply the initial condition \(y(0)=-1\text{:}\)
      🔗
      \begin{equation*} -1 = -\frac{33}{32} + C \quad\Longrightarrow\quad C = \frac{1}{32}. \end{equation*}
      Therefore, the specific solution is
      🔗
      \begin{equation*} y(x) = -\frac{33}{32}e^{-x/3} + \frac{1}{32} e^{x}. \end{equation*}
      🔗
    🔗
    🔗
    9.
    A nitric acid solution flows at a constant rate of 6 L/min into a large tank that initially held 200 L of a 0.5% nitric acid solution. The solution inside the tank remains stirred and flows out at a rate of 8 L/min. If the solution entering the tank is 20% nitric acid, determine the volume \(V(t)\) of nitric acid in the tank after \(t\) minutes.
    🔗
    Initially, \(V(0) = 0.005 \cdot 200\ \mathrm{L} = 1\ \mathrm{L}.\) The tank volume decreases because more flows out than in: \(200 - 2t\) liters after \(t\) minutes. The differential equation is
    🔗
    \begin{equation*} \frac{dV}{dt} = 1.2 \;-\; \frac{4}{100 - t}\,V. \end{equation*}
    1. Show that the solution to the IVP
      \begin{equation*} \frac{dV}{dt} = 1.2 - \frac{4}{100-t}\,V, \qquad V(0)=1, \end{equation*}
      obtained using an integrating factor, is
      \begin{equation*} V(t) = 0.4(100 - t) \;-\; \frac{39}{100^{4}}(100 - t)^{4}. \end{equation*}
      🔗
      🔗
    2. After how many minutes \(T\) is the tank empty?
      🔗
      🔗
    3. Since the tank empties at time \(T\text{,}\) one expects \(V(T)=0\) and that the model is valid for \(0 \le t \le T\text{.}\) Does the graph of \(V(t)\) support this?
      🔗
      🔗
    Solution.
    1. Rewrite the ODE in linear form:
      🔗
      \begin{equation*} \frac{dV}{dt} + \frac{4}{100 - t}V = 1.2. \end{equation*}
      The integrating factor is
      🔗
      \begin{align*} I(t) \amp = \exp\!\left(\int \frac{4}{100 - t}\,dt\right) \\ \amp = \exp\!\left(-4\ln(100 - t)\right) \\ \amp = (100 - t)^{-4}. \end{align*}
      Multiply both sides of the ODE by \((100 - t)^{-4}\text{:}\)
      🔗
      \begin{equation*} (100 - t)^{-4}\frac{dV}{dt} + (100 - t)^{-5}(-4)V = 1.2(100 - t)^{-4}. \end{equation*}
      The left-hand side is the derivative of the product:
      🔗
      \begin{equation*} \frac{d}{dt}\!\left( (100 - t)^{-4} V \right) = 1.2(100 - t)^{-4}. \end{equation*}
      Integrate both sides:
      🔗
      \begin{equation*} (100 - t)^{-4} V = \int 1.2(100 - t)^{-4}\,dt + C. \end{equation*}
      Let \(u = 100 - t\text{,}\) so \(du = -dt\text{.}\) Then
      🔗
      \begin{align*} \int 1.2(100 - t)^{-4}\,dt \amp = -1.2\int u^{-4}\,du \\ \amp = -1.2\left(\frac{u^{-3}}{-3}\right) \\ \amp = 0.4\,u^{-3} \\ \amp = 0.4(100 - t)^{-3}. \end{align*}
      Hence:
      🔗
      \begin{equation*} (100 - t)^{-4} V = 0.4(100 - t)^{-3} + C. \end{equation*}
      Multiply both sides by \((100 - t)^{4}\text{:}\)
      🔗
      \begin{equation*} V(t) = 0.4(100 - t) + C(100 - t)^{4}. \end{equation*}
      Apply \(V(0) = 1\text{:}\)
      🔗
      \begin{equation*} 1 = 0.4(100) + C(100)^{4} = 40 + C\cdot 100^{4}. \end{equation*}
      Therefore,
      🔗
      \begin{equation*} C = -\frac{39}{100^{4}}. \end{equation*}
      The solution is thus
      🔗
      \begin{equation*} V(t) = 0.4(100 - t) \;-\; \frac{39}{100^{4}}(100 - t)^{4}. \end{equation*}
      🔗
    2. The tank is empty when its total volume becomes zero. Since the tank starts with 200 L and loses 2 L/min (because outflow is 8 L/min and inflow is 6 L/min), we solve:
      🔗
      \begin{equation*} 200 - 2T = 0 \Longrightarrow T = 100\ \text{minutes}. \end{equation*}
      Thus the tank becomes empty after \(T = 100\) minutes.
      🔗
      🔗
    3. Evaluate \(V(t)\) at \(t=100\text{:}\)
      🔗
      \begin{equation*} V(100) = 0.4(100 - 100) - \frac{39}{100^{4}}(100 - 100)^{4} = 0, \end{equation*}
      as expected. On the interval \(0 \le t \le 100\text{,}\) the expression \(100 - t\) remains nonnegative, so the graph of \(V(t)\) looks physically reasonable: it starts at \(V(0)=1\text{,}\) rises initially (since the inflow is very concentrated relative to tank volume), and eventually decreases as the tank drains. The graph approaches 0 smoothly as \(t\to 100\text{.}\) The behavior is fully consistent with the mixing model.
      🔗
      🔗

    10.
    At noon, a tank contains 100 L of solution in which 10 kg of chemical is dissolved. Solution containing 2 kg of the chemical per liter flows into the tank at 5 L/min. The mixture is well-stirred and drawn off at a rate of 4 L/min. If the tank holds 500 L, when does it overflow? If \(t\) measures minutes after noon, find the mass of chemical \(x(t)\) (in kg) in the tank at time \(t\text{.}\) (Set up and solve a linear initial-value problem.)
    🔗
    Solution.
    Step 1. Tank volume.
    🔗
    Inflow is 5 L/min, outflow is 4 L/min, so volume increases at \(1\ \mathrm{L/min}\text{.}\)
    🔗
    \begin{equation*} V(t) = 100 + t. \end{equation*}
    The tank overflows when \(V(t)=500\text{:}\)
    🔗
    \begin{equation*} 100 + t = 500 \quad\Longrightarrow\quad t = 400\ \text{minutes}. \end{equation*}
    Step 2. Differential equation for mass of chemical.
    🔗
    Chemical entering: \((2\ \mathrm{kg/L})(5\ \mathrm{L/min}) = 10\ \mathrm{kg/min}\text{.}\)
    🔗
    Chemical leaving: concentration in tank is \(x(t)/V(t)\text{,}\) so outflow chemical rate is
    🔗
    \begin{equation*} \frac{x(t)}{100+t}\,(4\ \mathrm{L/min}). \end{equation*}
    Therefore,
    🔗
    \begin{equation*} \frac{dx}{dt} = 10 \;-\; \frac{4}{100+t}\,x(t), \qquad x(0)=10. \end{equation*}
    Step 3. Solve the linear ODE using an integrating factor.
    🔗
    Write in standard form:
    🔗
    \begin{equation*} x' + \frac{4}{100+t}x = 10. \end{equation*}
    Integrating factor:
    🔗
    \begin{align*} \mu(t) \amp = \exp\!\left(\int \frac{4}{100+t}\,dt\right) \\ \amp = (100+t)^{4}. \end{align*}
    Multiply the ODE by \((100+t)^{4}\text{:}\)
    🔗
    \begin{equation*} \frac{d}{dt}\!\bigl((100+t)^{4} x\bigr) = 10(100+t)^{4}. \end{equation*}
    Integrate:
    🔗
    \begin{align*} (100+t)^{4} x \amp = 10\int (100+t)^{4}\,dt + C \\ \amp = 10\left(\frac{(100+t)^{5}}{5}\right) + C \\ \amp = 2(100+t)^{5} + C. \end{align*}
    Divide by \((100+t)^{4}\text{:}\)
    🔗
    \begin{equation*} x(t) = 2(100+t) + C(100+t)^{-4}. \end{equation*}
    Use \(x(0)=10\text{:}\)
    🔗
    \begin{equation*} 10 = 200 + C(100)^{-4} \quad\Longrightarrow\quad C = -190\cdot 100^{4}. \end{equation*}
    Final expression:
    🔗
    \begin{equation*} x(t) = 2(100+t) \;-\; 190\left(\frac{100}{100+t}\right)^{4}. \end{equation*}
    Answer summary:
    🔗
    • The tank overflows at \(t=400\) minutes after noon.
      🔗
    • The mass of chemical is \(x(t) = 2(100+t) -190\left(\frac{100}{100+t}\right)^{4}.\)

    A tank initially contains 200 gallons of pure water. Saltwater with concentration 3 ounces of salt per gallon enters at \(\tfrac12\) gallon per minute. The mixture drains at the same rate. Additionally, a filtering device removes half the salt contained in one gallon of the mixture per minute.
    🔗
    Let \(y(t)\) be the number of ounces of salt in the tank at time \(t\) minutes.
    🔗
    1. Find the initial-value problem (IVP) modeling this situation.
      🔗
      🔗
    2. Use technology to sketch a slope field (\(-200 \le t \le 1440,\; -80 \le y \le 800\)). Do any equilibrium solutions appear? If so, where?
      🔗
      🔗
    3. Solve the IVP for \(y(t)\text{.}\) Plot \(y(t)\) for the first 24 hours.
      🔗
      🔗
    4. Compute \(\displaystyle \lim_{t\to\infty} y(t)\text{.}\)
      🔗
      🔗
    Solution.
    1. Modeling the IVP.
      🔗
      Inflow salt rate:
      🔗
      \begin{equation*} (3\ \mathrm{oz/gal})(\tfrac12\ \mathrm{gal/min}) = 1.5\ \mathrm{oz/min}. \end{equation*}
      Outflow salt rate: volume stays 200 gallons, so concentration is \(y/200\text{,}\) and outflow rate \(\tfrac12\) gal/min gives
      🔗
      \begin{equation*} \frac{y}{200}\cdot \tfrac12 = \frac{y}{400}\ \mathrm{oz/min}. \end{equation*}
      Filtering device removes “half the salt in one gallon per minute”:
      🔗
      \begin{equation*} \frac12\cdot \frac{y}{200} = \frac{y}{400}\ \mathrm{oz/min}. \end{equation*}
      Total removal rate is \(y/400 + y/400 = y/200\text{.}\)
      🔗
      Therefore, the initial-value problem is
      🔗
      \begin{equation*} \frac{dy}{dt} = 1.5 - \frac{y}{200}, \qquad y(0)=0. \end{equation*}
      🔗
    2. Equilibrium solution.
      🔗
      Set \(y'=0\text{:}\)
      🔗
      \begin{equation*} 1.5 - \frac{y}{200} = 0 \quad\Longrightarrow\quad y = 300. \end{equation*}
      Thus an equilibrium appears at \(y=300\) ounces. The slope field (using any technology) will show horizontal segments near \(y=300\text{.}\)
      🔗
      🔗
    3. Solve the IVP.
      🔗
      Rewrite:
      🔗
      \begin{equation*} y' + \frac{1}{200}y = 1.5. \end{equation*}
      Integrating factor:
      🔗
      \begin{equation*} \mu(t) = e^{t/200}. \end{equation*}
      Multiply through:
      🔗
      \begin{equation*} \frac{d}{dt}\!\bigl(e^{t/200} y\bigr) = 1.5 e^{t/200}. \end{equation*}
      Integrate:
      🔗
      \begin{align*} e^{t/200} y \amp = 1.5(200)\,e^{t/200} + C \\ \amp = 300 e^{t/200} + C. \end{align*}
      Divide by \(e^{t/200}\text{:}\)
      🔗
      \begin{equation*} y(t) = 300 + C e^{-t/200}. \end{equation*}
      Use \(y(0)=0\text{:}\)
      🔗
      \begin{equation*} 0 = 300 + C \quad\Longrightarrow\quad C=-300. \end{equation*}
      Final expression:
      🔗
      \begin{equation*} y(t) = 300\bigl(1 - e^{-t/200}\bigr). \end{equation*}
      A 24‑hour plot (over \(0 \le t \le 1440\)) shows a rising curve approaching 300.
      🔗
      🔗
    4. Long-term behavior.
      🔗
      Compute the limit:
      🔗
      \begin{align*} \lim_{t\to\infty} y(t) \amp = 300\left(1 - \lim_{t\to\infty} e^{-t/200}\right) \\ \amp = 300. \end{align*}
      The amount of salt approaches the equilibrium value 300 ounces.
      🔗
